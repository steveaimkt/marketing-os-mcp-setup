# 클립 1-4 실습 · Firecrawl MCP

## 목표 산출물
- API 키 등록 완료
- 본인이 분석할 경쟁사 사이트 1개를 마크다운으로 추출

## 단계별 명령

### Step 1. API 키 발급
```
1. https://firecrawl.dev 가입
2. Dashboard → API Keys → Create
3. 키 복사 (한 번만 표시됨)
```

### Step 2. .env 설정
```bash
FIRECRAWL_API_KEY=fc-xxxxxxxxxxxx
```

### Step 3. .mcp.json 추가
```json
{
  "mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "${FIRECRAWL_API_KEY}"
      }
    }
  }
}
```

### Step 4. 헬스 체크
```bash
claude
> /mcp   # firecrawl ✅
> "https://example.com 페이지를 마크다운으로 가져와줘"
# → 페이지 본문이 마크다운으로 응답
```

## 마케터 5패턴 적용 · 경쟁사 분석

```
[역할] 너는 K-뷰티 D2C 카테고리 분석가야.

[입력] 경쟁사 도메인 3개:
  - https://medicube.co.kr
  - https://torriden.com
  - https://numbuzin.com

[산출물] 각 도메인의 (1) 메인 신제품 1개 (2) 가격 (3) 핵심 USP 3가지를 표로.

[제약] 마크다운 표 형식. 한국어. 가격은 원화 정수.

[검증] 추출 실패한 도메인이 있으면 사유 표시.
```

## 트러블슈팅

| 증상 | 해결 |
|---|---|
| Rate limit | `claude mcp restart firecrawl` 후 60초 대기 |
| JS 페이지 빈 응답 | "waitFor 3000ms 옵션으로 다시 가져와줘" 명시 |
| 한글 깨짐 | "formats=['markdown']로 다시" |
