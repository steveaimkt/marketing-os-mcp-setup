## 🧠 마케터 활용법 8가지

### A. 채널별 성과 추적 (가장 기본)
```
"지난 7일 채널별 세션·전환·전환율"
→ run_report (sessionDefaultChannelGroup 차원)
```

### B. 콘텐츠 (랜딩 TOP 10)
```
"지난 28일 랜딩 페이지 TOP 10 + 이탈률"
→ run_report (landingPagePlusQueryString 차원)
```

### C. 추세 비교 (이번 주 vs 지난 주)
```
"이번 주 vs 지난 주 오가닉 트래픽 변화율"
→ batch_run_reports (기간 2개 동시)
```

### D. 신규 vs 재방문 (LTV 분석)
```
"newVsReturning 차원으로 전환율 비교"
→ run_report (newVsReturning 차원)
```

### E. 캠페인 추적 (UTM)
```
"지난달 utm_source 별 세션·전환"
→ run_report (sessionSource, sessionMedium, sessionCampaignName)
```

### F. 디바이스 분포 (모바일 비중)
```
"모바일·데스크탑·태블릿 전환율 차이"
→ run_report (deviceCategory 차원)
```

### G. 지역 분포 (국가별 매출)
```
"국가별 매출 TOP 10"
→ run_report (country, city 차원)
```

### H. 실시간 활성 (캠페인 직후 모니터링)
```
"지금 활성 사용자 몇 명, 어떤 페이지?"
→ get_realtime_data
```

## 💡 자주 묻는 질문

**Q. Universal Analytics (구버전) 도 가능한가요?**  
A. 안 됨. **GA4 전용**. UA 는 2023-07-01 부로 데이터 수집 중단됨.

**Q. 서비스 계정 권한 부여 후 바로 작동하나요?**  
A. **평균 5~10분 대기** 필요. `PERMISSION_DENIED` 응답 시 10분 후 재시도.

**Q. 커스텀 차원·이벤트 매개변수도 조회 가능한가요?**  
A. 네. GA4 관리 → 맞춤 정의에 등록된 차원·지표는 모두 자연어로 호출 가능. 단, 표준 차원과 이름 충돌 주의.

**Q. 광고 데이터까지 합쳐서 ROAS 계산도 되나요?**  
A. GA4 단독으로는 어려움. Part 6 Meta·Google Ads MCP 와 결합한 통합 에이전트 (Part 8 `cross-channel-roas`) 권장.

**Q. 데이터 응답이 비어 있어요.**  
A. 3가지 확인: ① GA4 속성에 실제 트래픽 데이터 존재 / ② 속성 ID 9자리 숫자만 (접두사 `properties/` X) / ③ 서비스 계정 권한 활성화 완료 (10분 대기)
