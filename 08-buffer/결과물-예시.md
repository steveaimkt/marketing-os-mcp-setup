## 🧠 마케터 활용법 6가지

### A. 동일 콘텐츠 4채널 동시 발행 (가장 자주)
```
"이 글 4채널에 채널별 톤 최적화해서 예약"
→ Claude 가 채널별 분기 작성 + 예약
```

### B. 콘텐츠 캘린더 1주일치 일괄
```
"다음 5개 블로그 글을 월~금 매일 09시 LinkedIn에 예약"
→ 5회 schedule_post 자동 실행
```

### C. 캠페인 D-7 카운트다운
```
"5월 30일 신상품 출시 카운트다운 글 5개를 D-7부터 매일 19시"
→ Claude 가 카피 5개 생성 + 5회 예약
```

### D. 휴일·이벤트 자동 콘텐츠
```
"내년 1월 ~ 12월 매월 1일 09시 본인 브랜드 인사 글 예약"
→ 12회 schedule_post 일괄
```

### E. 인기 글 재포스팅
```
"지난 3개월 좋아요 가장 높은 인스타그램 글 5개를 다시 예약"
→ get_top_posts + reschedule
```

### F. 채널별 성과 추적 후 자동 재배분
```
"LinkedIn 좋아요 가장 높은 후크 패턴 분석 → 다음 글에 자동 적용"
→ get_post_stats + Part 5 social-copy-writer 에이전트
```

## ⏱️ 시간 비교

| 작업 | Before (수동) | After |
|---|---|---|
| 채널별 카피 4개 작성 | 60분 | 1~2분 |
| 채널별 글자 수·해시태그 최적화 | 30분 | 자동 |
| 4개 채널 각각 예약 | 20분 | 자동 |
| **신규 글 1건 4채널 발행** | **약 1시간 50분** | **2~3분** |

매일 1건 발행 시 연 환산: **약 600시간 절감**.

## 🎯 본인 데이터로 따라하기 (5분)

1. `buffer.com` 가입 → Settings → Access Tokens → 발급
2. `.env`에 `BUFFER_ACCESS_TOKEN=1/xxx` 등록
3. Buffer 에 LinkedIn/Twitter/Instagram/Facebook 채널 연결
4. Claude Code:
   ```
   이 글을 4채널에 내일 09시 예약: <블로그 URL>
   ```
5. 1~2분 대기 → 예약 4건 응답

## 💡 자주 묻는 질문

**Q. Buffer 무료 플랜으로 가능?**  
A. 무료 플랜은 3 채널 + 10개 예약. 마케터 일상 사용은 Essentials ($6/월) 권장.

**Q. 한국어 캡션도 잘 처리?**  
A. UTF-8 OK. 다만 Twitter 280자 제한은 한글이 2배 가중치 → Claude 자동 길이 조절.

**Q. 카카오톡·네이버 블로그도?**  
A. Buffer 공식 미지원. 별도 MCP 필요.

**Q. 이미지·동영상 첨부도?**  
A. Buffer API 지원. Claude 가 URL 전달 → 자동 업로드.

**Q. 예약 후 수정·취소 가능?**  
A. `update_post` / `delete_post` 도구 활용. "내일 09시 예약 글 본문 바꿔줘" 자연어 OK.
