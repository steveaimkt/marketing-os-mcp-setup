# 클립 2-1 실습 · Figma MCP 연결 (Claude.ai 통합)

## 목표 산출물

- Claude.ai > Settings > Integrations 에서 Figma Connect 완료
- Claude Code 안에서 `claude mcp list | grep figma` → ✓ Connected 확인
- 본인 Figma 파일 1개에서 카피·컴포넌트·색상 추출 검증

## 사전 준비물

```
□ Claude.ai 계정 (Claude Code 와 동일 계정 권장)
□ Figma 계정 (무료 또는 유료 OK)
□ 분석할 Figma 파일 1개 (본인 접근 권한)
□ Chrome 또는 Safari
```

> ⚡ Personal Access Token 발급·Figma Desktop 플러그인 설치·WebSocket 서버·Bun 런타임 **모두 불필요**. Claude.ai 통합으로 한 번 클릭.

## 트리거 1줄

```
Figma MCP 설치하자
```

→ `/mcp설치-figma` 스킬 자동 호출.

## 단계별 명령

### Step 1 · Claude.ai > Integrations 연결 (사용자 1분)

```
1. 새 탭에서 https://claude.ai 접속
   (Claude Code 와 같은 계정으로 로그인)

2. 우상단 프로필 ⓘ 또는 ⚙ > "Settings" 클릭

3. 좌측 메뉴 "Integrations" (통합) 클릭

4. 통합 목록에서 "Figma" 찾기
   → 우측 "Connect" 버튼 클릭

5. 브라우저 자동으로 Figma OAuth 페이지 열림
   - Figma 로그인 (이미 로그인 상태면 패스)
   - "Allow" 또는 "Authorize Claude" 클릭

6. Claude.ai > Integrations 화면으로 자동 복귀
   → Figma 항목에 ✓ Connected 표시 확인
```

### Step 2 · Claude Code 헬스 체크 (자동 30초)

```bash
# Claude Code 안에서
claude mcp list | grep -i figma
# → claude.ai Figma: https://mcp.figma.com/mcp - ✓ Connected
```

`✓ Connected` 가 아니면:
- Claude.ai 와 Claude Code 의 로그인 계정이 같은지 확인
- Claude Code 완전 종료 후 재시작

### Step 3 · 첫 결과물 (사용자 30초 + Claude 자동 8초)

#### 3.1 본인 Figma 파일 URL 준비

```
1. figma.com 에서 본인 파일 열기
2. 상단 주소창 URL 복사
   (예: https://www.figma.com/design/abc123/Spring-Campaign-2026)
3. (선택) 본인 파일이 없으면 Figma Community 의 무료 템플릿 활용:
   - figma.com/community → "Landing Page" 검색 → "Duplicate" 클릭 → 본인 계정에 복제
```

#### 3.2 Claude Code 명령

```
다음 Figma 파일의 모든 텍스트 + 컴포넌트 추출해줘:
<위에서 복사한 URL>
```

→ 8초 대기 → 종합 분석 응답:
- 페이지 / 프레임 / 컴포넌트 / 텍스트 요소 갯수
- 모든 헤드라인·CTA·본문 텍스트 추출
- 컬러 시스템 (Primary·Secondary·Accent)
- 폰트 (Bold·Regular 크기 분포)
- 이미지 export 가능 갯수

자세히는 [결과물-예시.md](결과물-예시.md).

## 마케터 5패턴 적용 예시

```
[역할] D2C 브랜드의 콘텐츠 마케터

[입력] 디자이너가 보낸 Figma 캠페인 시안 URL

[산출물]
1. 헤드라인 10개 + 톤별 3패턴씩 (감성·실용·유머 = 30패턴)
2. 컬러 시스템 markdown (Primary·Secondary·Accent)
3. 이미지 자산 export 목록
4. 추천 조합 (인스타: 감성 / 광고: 실용 / 콘텐츠: 유머)

[제약]
- 한국어 자연어 출력
- 카피 변형 시 본인 브랜드 톤 유지

[검증]
- 추출 텍스트 갯수 ≥ 시안의 실제 텍스트 갯수
- 컬러 hex 코드 정확도 100%
```

## 트러블슈팅

| 증상 | 원인 | 해결 |
|---|---|---|
| Claude.ai > Integrations 에 Figma 없음 | 지역·계정 권한 차이 | claude.ai/integrations 직접 접근 시도 |
| `mcp__claude_ai_Figma__*` 도구 안 보임 | 통합 미연결 또는 Claude Code 재시작 안 함 | Settings > Integrations 에서 Connect 확인 + Claude Code 재시작 |
| `Permission denied` (특정 파일) | 본인 접근 권한 없는 파일 | Figma 에서 본인 계정에 공유 권한 추가 |
| FigJam 보드 추출 안 됨 | 일반 Figma 도구 호출 | `get_figjam` 도구 명시적 사용 |
| 한국어 카피 깨짐 | 폰트 한글 미지원 | Pretendard / Spoqa Han Sans 설정 확인 |
