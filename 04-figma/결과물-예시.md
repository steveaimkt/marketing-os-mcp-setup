# 클립 2-1 결과물 예시 · Figma 디자인 → 콘텐츠 자동 추출

> Figma MCP (Claude.ai 통합) 설치 직후 마케터가 받게 되는 첫 결과물. OAuth 자동 인증.

## 한 줄 요약

본인 Figma 파일 URL 1줄 → Claude가 디자인 컴포넌트·텍스트·이미지 자동 추출 → 마케팅 카피·디자인 시스템 가이드 자동 생성.

## 📊 결과물 2 · 카피 변형 자동 생성

```
사용자: "위 헤드라인 10개를 톤별 3패턴씩 변형 (감성/실용/유머)"

응답:

| 원본 | 감성 | 실용 | 유머 |
|---|---|---|---|
| 봄, 다시 시작 | 봄이 내게 살짝 인사를 건넸다 | 봄철 피부 회복 3주 솔루션 | 봄, 너 거기 있었구나 |
| 한 번 바르면 매일 새로워지는 피부 | 매일 아침, 거울이 미소짓는다 | 1회 사용으로 효과 체감 | 거울이 자꾸 물어봐요 |
| 10일 후 거울 앞에서 놀랄 변화 | 10일이 만든 작은 기적 | 임상 10일 효과 92% | 10일만에 친구가 누구냐고 |

📌 추천 조합 (Claude 자동 도출)
  · 인스타: 감성 톤 (인게이지먼트 추정 +34%)
  · 광고: 실용 톤 (전환율 추정 +18%)
  · 콘텐츠: 유머 톤 (공유율 추정 +47%)
```

## 🧠 마케터 활용법 6가지

### A. 디자인 → 카피 자동 추출
```
"이 Figma 파일의 모든 카피 추출 → 톤별 변형"
→ get_design_context + Claude 변형 생성
```

### B. 디자인 시스템 가이드 자동
```
"이 Figma 파일의 컬러·폰트·간격 가이드 markdown으로"
→ get_design_context + Claude 정리
```

### C. 신규 디자인 자동 생성
```
"Spring Campaign 2026 톤으로 여름 캠페인 디자인 컨셉 만들어줘"
→ generate_design + 톤 학습
```

### D. 디자인 → React/HTML 코드
```
"이 프레임을 React 컴포넌트 코드로 변환"
→ get_design_context + Code Connect
```

### E. 이미지 자산 일괄 추출
```
"이 페이지의 모든 이미지를 webp로 export"
→ download_design (export 형식 지정)
```

### F. 디자인 변경 추적
```
"이 파일의 지난주 vs 이번주 변경 사항"
→ get_metadata + diff 분석
```

## 💡 자주 묻는 질문

**Q. Figma 무료 플랜으로?**  
A. 가능. 단 공유 권한이 있어야 Claude.ai 접근.

**Q. 비공개 파일도?**  
A. OAuth 인증 후 본인이 접근 가능한 모든 파일. 팀 파일도 권한 있으면 OK.

**Q. FigJam 보드도?**  
A. 별도 get_figjam 도구로 가능. 다이어그램 추출 OK.

**Q. 디자인 직접 수정도?**  
A. use_figma 도구로 가능. 단 신중하게 적용 (실수로 디자인 손상 방지).

**Q. 한국어 폰트·디자인도?**  
A. Pretendard, Spoqa Han Sans 등 자동 인식. 한국어 카피 정상 추출.
