### 🎬 [0:25~1:30] Before/After · 외주 4시간 → 10분

〔슬라이드: 02 외주 비용·시간 비교〕

**🎙️ 강사 발화:**

> "마케팅 캠페인마다 반복되는 UI(유아이·사용자 인터페이스) 페이지가 있어요. 로그인·회원가입·캠페인 배너·SNS 카드뉴스·가격 비교표·모바일 온보딩. 매 캠페인마다 새로 그려야 합니다.
>
> 현재 마케터 옵션이 3가지인데 모두 비효율적이에요.
>
> **옵션 1 · 디자이너 외주** · 한 건당 반나절~며칠 + 30~50만원. 수정 요청마다 다시 대기.
> **옵션 2 · 사내 디자이너** · 회의 → 의뢰 → 재수정 사이클. 캠페인 일정 잡아먹음.
> **옵션 3 · 마케터 직접** · Figma·Photoshop 배워야. 1건에 2~3시간.
>
> **Before(전)** · 로그인 페이지 1건 만들기 = 외주 반나절 + 30~50만원.
>
> **After(후)** · Claude에 한 줄 '다크모드 로그인 페이지 만들어줘. 이메일·비밀번호·소셜 로그인 3종.' Figma 파일에 페이지가 직접 생성됩니다. **10분 + 0원.**
>
> 약 24배 단축. 외주비 절감 + 즉시 수정 가능. **'색상만 빨간색으로 바꿔줘'** 한 줄로 1초 수정."

〔→ 03 5 시나리오〕

### 🎬 [1:30~3:00] 5 시나리오 · 직접 만들 수 있는 UI 5가지

〔슬라이드: 03 5 시나리오 카드〕

**🎙️ 강사 발화:**

> "마케터가 직접 만드는 UI 5가지입니다.
>
> **A. 로그인·회원가입 페이지 ★** · '다크모드 로그인 페이지, 이메일·비밀번호·소셜 3종 만들어줘' 5~10분.
>
> **B. 캠페인 배너** · '스킨케어 봄 캠페인 배너 1200×400, 핑크 톤' 3~5분. 광고용·사이트 상단용 즉시.
>
> **C. SNS 카드뉴스 ★** · '인스타 1080×1080 카드 5장 만들어줘' 10분. 캐러셀 포스트 일괄.
>
> **D. 가격 비교 표** · '3단 pricing 카드 (Free/Pro/Enterprise) 만들어줘' 5~10분.
>
> **E. 모바일 앱 화면** · '온보딩 3단계 모바일 화면' 약 3분.
>
> 시작 순서 · 처음이면 A 로그인 또는 B 배너부터 (가장 자주 쓰는 케이스). 익숙해지면 C·D·E. **24섹션 상세페이지**처럼 복잡한 건 별도 Figma 플러그인이 따로 있어요 (Part 5에서).
>
> 분석도 가능합니다 · 기존 Figma 파일의 색상·폰트·간격을 한 줄로 추출하거나, 디자인을 HTML/CSS 코드로 변환. 30분 → 2분."

〔→ 04 작동 원리〕

### 🎬 [3:00~4:00] 작동 원리 · 한 줄 명령이 다크모드 로그인 페이지가 되기까지

〔슬라이드: 04 작동 흐름 · start → 라우터 → 메뉴 → 자동 실행 → 결과물〕

**🎙️ 강사 발화:**

> "실제로 어떻게 작동하는지 원리를 짚어드릴게요. 설치가 끝난 상태라면, Claude한테 단어 하나만 입력하면 됩니다.
>
> **`start`** ← 슬래시 명령 아니에요. 그냥 'start' 또는 '시작' 한 단어. 본 강의가 미리 만들어둔 라우터가 자동으로 켜집니다.
>
> 그러면 Figma(피그마)가 설치되어 있는 걸 자동으로 감지해서, **작업 가능 업무 5개 메뉴**를 띄워줍니다 · 카피 추출 + 톤 변형·디자인 시스템 가이드·디자인 → 코드 변환·이미지 자산 export·신규 디자인 컨셉. 본인은 '**1**' 한 번만 누르면 돼요.
>
> Claude가 본인 Figma 파일에서 헤드라인·CTA·본문 텍스트를 자동 추출하고, 감성/실용/유머 3톤으로 각각 10개씩 변형 카피를 만들어줍니다. 약 30초.
>
> 결과물은 Figma 파일에 직접 생성된 다크모드 로그인 페이지입니다 · 이메일 입력·비밀번호 입력·"로그인" 버튼·소셜 로그인 3개·"회원가입" 링크가 한 화면에 자동으로 배치됩니다. (실제 실행 화면은 별도 실습 영상에서 보여드립니다.)
>
> 수정이 필요하면 한 줄 더 · '버튼 색을 파란색으로' · '소셜 로그인을 아래로 옮겨줘' · '비밀번호 찾기 링크 추가' · 각 1초 안에 반영됩니다.
>
> 같은 패턴으로 회원가입 페이지·캠페인 배너·SNS 카드 5장도 즉시. 캠페인 한 번에 필요한 모든 비주얼 자산이 30분 안에 끝납니다."

### 🎬 [4:00~4:35] 마케터 가치 3가지

〔슬라이드: 05 가치 3 카드〕

**🎙️ 강사 발화:**

> "Figma MCP가 마케터한테 주는 가치 3가지입니다.
>
> **첫째, 외주비가 0원이 됩니다** · 로그인 페이지·배너 1건당 30~50만원이 사라져요. 매주 캠페인 발행 마케터라면 월 100~200만원.
>
> **둘째, 즉시 수정 가능** · '색상 빨간색으로' 한 줄로 1초. 디자이너 회의·재수정 사이클 없음.
>
> **셋째, 마케터 디자인 자립** · 디자이너 의존도 0. 신입 마케터도 첫날부터 캠페인 비주얼 생산.
>
> 한 줄로 정리하면 · **디자이너를 옆에 둔 채로 마케터 혼자 일한다.**"
