## 🎯 본인 데이터로 따라하기 (10분)

### Meta Ads (1분 · Meta 공식 hosted MCP)
1. 터미널: `claude mcp add --transport http --scope user meta-ads https://mcp.facebook.com/ads`
2. Claude Code 재시작 → "Meta 광고 캠페인 보여줘" 입력 → 브라우저 OAuth → 5 scope 허용
3. .env 변수 발급·등록 불필요. 토큰 자동 갱신.

### Google Ads (5분 + 1~2일 승인 대기 · Google 공식 MCP)
1. `ads.google.com/aw/apicenter` → Developer Token 신청 (5분)
2. 1~2일 후 승인 이메일 → 22자 Developer Token 복사
3. `.env` 에 `GOOGLE_ADS_DEVELOPER_TOKEN` 한 줄만 등록
4. ADC 는 GA4 와 공유 (이미 발급됨, 별도 작업 X)
5. 첫 호출: "내 Google 광고 계정 보여줘" → `list_accessible_customers` 자동 실행

## 💡 자주 묻는 질문

**Q. 광고 직접 수정·집행도 가능?**  
A. 가능. 단 안전을 위해 marketing-os 기본은 **읽기 전용** 설정. 예산 수정 등은 사용자 명시적 승인 필요.

**Q. 두 플랫폼 데이터 직접 비교 가능?**  
A. CTR/CPA/ROAS 등 표준 지표 비교 가능. 단 각 플랫폼 정의 차이 (예: 전환 윈도우) 주의 필요.

**Q. 한국 광고 계정도 OK?**  
A. 네. 통화 자동 인식 (KRW). 한국어 광고 텍스트 정상 처리.

**Q. Meta 토큰 만료?**  
A. Meta 공식 hosted MCP 는 OAuth 2.1 표준 · 토큰은 Claude.ai 가 자동 갱신. 사용자가 수동 재발급 불필요. (이전 npm 방식의 System User Token / 일반 User Token 60일 만료 이슈 모두 해소됨)

**Q. Google Ads Developer Token 발급 거절?**  
A. Test Account 사용 가능 (운영 데이터 X). 또는 Manager Account 추가 후 신청 재시도.
