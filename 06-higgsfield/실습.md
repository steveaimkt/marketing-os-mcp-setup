# 클립 2-3 실습 · Higgsfield MCP · 모델 이미지 1장 + 영상 1편 자동 생성 (Marketor 케이스)

## 실습 한 줄 요약

`/mcp설치-higgsfield` 스킬로 5~7분 안에 설치 → **Claude 가 포토 디렉터로 4단계 (기획 → 크레딧·모델 안내 → 생성 → 수정)** → 모델 키비주얼 1장 + 8초 광고 영상 1편 자동 생성.

> 💡 **사전 기획이 크레딧 절감의 핵심** · Claude 디렉팅 없이 미드저니에 직접 던지면 결과가 들쭉날쭉 → 재생성 → 크레딧 낭비. Claude 가 먼저 앵글·조명·색감·모델 프로필 10항목을 기획해주니까 한 번에 원하는 결과 → 크레딧 절약.

> 💡 **모델 이미지 먼저, 영상은 그 결과를 시작 프레임으로 체이닝** · 인물이 들어가는 영상은 이미지부터 톤 확정 후 영상으로. 영상은 크레딧이 커서 한 번에 잘 나와야 함.

> 📖 **실제 진행 케이스** : Marketor Nourishing Face Cream → [`결과물-예시.md`](결과물-예시.md) 참조 (실제 진행 로그 + Job ID + 크레딧 차감 기록).

## 사전 준비

```
□ Node.js 18 이상            → 터미널에서 `node --version`
□ Higgsfield 무료 계정       → higgsfield.ai 가입 (무료)
□ 본인 워크스페이스 1개      → 가입 시 자동 생성
□ Chrome 또는 Safari        → OAuth 인증용 브라우저
```

본 실습은 모두 무료 (가입 시 무료 크레딧 50 부여 · 이미지 1장 1~2 크레딧 / 5초 영상 5~10 크레딧).

## 4단계 작동 흐름 (모든 명령에 동일 적용)

```
사용자 한 줄 명령
   ↓
[단계 1] ★ Claude 프롬프트 기획 (포토 디렉터 역할)
  · 앵글·조명·구도·색감·제품 배치·카피 위치를 자동 디렉팅
  · 한국어 한 줄 → 영문 완성 프롬프트로 변환
   ↓
[단계 2] ★ 크레딧·모델 안내
  · "이 작업은 약 X 크레딧 소진 예정" 표시
  · "이 작업에는 모델 Y 가 적합" 추천
  · 잔여 크레딧 비교 후 진행 여부 확인
   ↓
[단계 3] 힉스필드 호출 → 이미지·영상 생성
   ↓
[단계 4] 결과 확인 → "색감 더 따뜻하게" 한 줄로 즉시 수정 루프
```

→ 단계 1·2 가 잘 되면 단계 4 재생성 횟수가 줄어듭니다. **크레딧 절감의 핵심.**

## 실습 1 · 모델 키비주얼 이미지 1장 (약 1분, 약 0.24~3 크레딧)

> 💡 영상에 인물이 들어가면 **모델 사진 먼저 → 영상 시작 프레임으로 체이닝** 하는 게 안정적. 영상은 크레딧이 크니까 한 번에 잘 나와야 함.

### 사용자 명령

```
화장품 신제품 모델 키비주얼 만들어줘.

[제품] Marketor Nourishing Face Cream (frosted glass jar, white lid, beige label)
[채널] 인스타그램 스토리·릴스 (9:16)
[컨셉] M1 에디토리얼 클로즈업 (잡지 화보 톤)
[모델] 20대 후반 한국 여성, 글로우 클린 피부, 어깨 자연 웨이브 헤어
[톤] 베이지·sage·아이보리 + 따뜻한 아침 자연광
[연출] 모델이 잔을 쇄골 근처에서 부드럽게 받쳐 듦
[카피 공간] 좌상단 1/3 비워둠
```

### 단계 1 · Claude 프롬프트 기획 (포토 디렉터)

Claude 가 위 한 줄을 받아 다음 정보를 자동 보강 :

```
📝 Claude 가 기획한 완성 프롬프트 (영문 변환)

Product: amber glass serum bottle 30ml, gold cap, glow serum label
Layout: 1:1 square, centered product on marble tray
Lighting: warm natural daylight from upper-left, soft shadow
Composition: 45-degree top-down angle, shallow depth of field
Color: beige-ivory gradient background, golden highlight on cap
Props: 2-3 dried rose petals, silk fabric edge
Style: luxury beauty editorial photography, photorealistic
Text: none (clean caption space at bottom)
Aspect ratio: 1:1
```

### 단계 2 · 크레딧·모델 안내

Claude 가 호출 직전에 표시 :

```
💰 사용 예정 안내

  · 모델 : Flux Pro (사실적 사진 · 럭셔리 뷰티 적합)
  · 예상 크레딧 : 1~2 크레딧
  · 잔여 크레딧 : 50 → 48 (예상)
  · 진행할까요? (y/n)
```

### 단계 3·4 · 생성 + (옵션) 수정

`y` → `mcp__higgsfield__generate_image` 호출 → 약 30~60초 후 결과 :

```
✅ 이미지 생성 완료
  - URL : https://higgsfield.ai/media/xxx.jpg
  - 모델 : Flux Pro
  - 크레딧 차감 : 2 (잔액 48)
```

수정 필요 시 한 줄 더 : `"색감을 더 따뜻하게 + 꽃잎 위치 오른쪽으로"` → 30초 안에 새 버전 (추가 1~2 크레딧).

## 실습 2 · 모델 이미지 → 8초 광고 영상 체이닝 (약 2~3분, 약 15~20 크레딧)

> 💡 실습 1 에서 만든 모델 이미지를 영상의 `start_image` 로 사용해 **인물·제품 외관 100% 유지**.

### 사용자 명령

```
실습 1 모델 이미지를 시작 프레임으로 8초 광고 영상 만들어줘.

[채널] Instagram Reels · TikTok · YouTube Shorts (9:16)
[장르] drama (잔잔한 톤)
[스토리보드 8초]
  0~2s : 잔잔한 아침 빛, 모델 표정 변화 없음 (느린 푸시-인 4°)
  2~4s : 흰 뚜껑이 천천히 살짝 들리며 옆으로 기울어짐 (초점 라벨→내용물)
  4~6s : 잔 안 크림 표면, 나무 스푼 위 크림 한 방울 윤기 (살짝 내려다보는 미세 틸트)
  6~8s : 햇살 플레어 좌측에서 부드럽게 번지며 마무리 (정지)
[제약] 텍스트 오버레이 없음, 자연스러운 모션
[추가] 바이럴 점수 예측도 함께
```

### 자동 호출 (Claude 자동)

```
mcp__higgsfield__generate_video
  prompt: <Claude 가 디렉팅한 8초 스토리보드 완성 프롬프트>
  model: seedance_2_0          # reference-driven, 제품 일관성 최강
  aspect_ratio: 9:16
  duration: 8
  resolution: 720p             # 1080p 시 비용 약 2배
  genre: drama
  medias: [{role: start_image, value: <실습 1 이미지 URL>}]
```

→ 크레딧 약 15~20 (720p · 8초) + 바이럴 점수 1 크레딧.

### 단계 1 · Claude 프롬프트 기획

```
📝 Claude 가 기획한 완성 프롬프트

Scene: young woman late 20s in front of bathroom mirror, soft morning light
Action: pumps serum twice onto back of hand, applies gently to face
Duration: 5 seconds
Aspect ratio: 9:16 vertical
Camera: close-up handheld, slight shake, focus on hand and face
Style: natural daily vlog, not advertorial, no captions, no music tags
Color grade: warm beige-ivory tone, matching previous product shot
Pacing: pump (0~1s) → drop on hand (1~2.5s) → face application (2.5~5s)
```

### 단계 2 · 크레딧·모델 안내

```
💰 사용 예정 안내

  · 모델 : Higgsfield Video (영상 전용 · 자연 모션 강함)
  · 예상 크레딧 : 5~10 크레딧 (5초 9:16)
  · 잔여 크레딧 : 48 → 38~43 (예상)
  · 바이럴 점수 예측 추가 : +1 크레딧
  · 합계 약 6~11 크레딧
  · 진행할까요? (y/n)
```

### 단계 3·4 · 생성 + 바이럴 점수

`y` → `mcp__higgsfield__generate_video` + `virality_predictor` 동시 호출 → 약 2~3분 후 :

```
✅ 영상 생성 완료
  - URL : https://higgsfield.ai/media/xxx.mp4
  - 길이 5초 · 9:16 · 1080p
  - 크레딧 차감 : 10 (잔액 38)

📊 바이럴 점수 예측 : 74 / 100
  - 강점 : 자연스러운 톤·따뜻한 색감·제품 노출 자연스러움
  - 개선점 :
    ① 첫 1초에 손동작 클로즈업 한 컷 추가
    ② 제품 라벨이 보이는 시간 0.5초 → 1초로
    ③ 마지막 0.5초 만족 표정 컷 추가
```

→ 점수 70+ 면 그대로 사용. 60 이하면 위 개선점을 명령에 반영 후 재생성 (영상이라 크레딧 더 큼 → 신중히).

## 결과 요약

| 항목 | 이미지 1장 | 영상 5초 |
|---|---|---|
| 소요 시간 | 약 30초~1분 | 약 2~3분 |
| 크레딧 소진 | 1~2 | 5~10 (+1 바이럴) |
| 외주 비용 비교 | 5~10만원 → 무료 | 50~200만원 → 무료 |
| 본인이 카메라·조명 용어 이해 필요 | ❌ | ❌ |
| Claude 가 디렉팅 | ✅ | ✅ |
| 크레딧·모델 사전 안내 | ✅ | ✅ |

**합쳐 약 7~12 크레딧 소진 (잔여 무료 50 중)**. 무료 크레딧 한도 안에서 캠페인 1건 (이미지 3~5장 + 영상 1~2개) 충분히 가능.

## 크레딧 절감 5가지 팁

1. **사전 기획에 시간 투자** · Claude 와 컨셉·톤·연출 충분히 정리 후 호출. 한 번에 원하는 결과 → 재생성 0
2. **이미지부터 시작, 영상은 검증 후** · 이미지로 톤·색감 확정 후 그 결과를 참고로 영상 명령
3. **모델 추천 그대로 따르기** · Claude 가 추천하는 모델이 비용 대비 품질 최적. 임의로 다른 모델 지정 X
4. **수정은 작게** · "색감 더 따뜻하게" 같은 한 줄 수정은 적은 크레딧. 전면 재기획은 큰 크레딧
5. **바이럴 점수 70+ 만 채택** · 영상은 크레딧 큼. 점수 낮으면 명령 다듬은 후 한 번에 다시

## 트러블슈팅

| 증상 | 원인 | 해결 |
|---|---|---|
| `OAuth 인증 실패` | 브라우저 팝업 차단 | 팝업 허용 + Claude Code 재시작 |
| `Insufficient credits` | 무료 크레딧 소진 | `show_plans_and_credits` 잔액 확인 + Plus 플랜 ($9/월) |
| `Generation failed` | 프롬프트 정책 위반 | Claude 가 자동 우회 프롬프트 제안 |
| 이미지 품질 낮음 | Schnell (빠른) 모델 사용됨 | "Flux Pro 명시" 또는 Claude 추천 따르기 |
| 영상이 어색·끊김 | 5초가 너무 짧거나 동작 복잡 | 동작 단순화 + 카메라 고정 명시 |
| 한국어 결과 어색 | 모델 한국어 약함 | Claude 가 자동 영문 변환 (단계 1) |
| 비용 폭증 | 영상 여러 번 재생성 | 영상은 1회 신중히 + 점수 70+ 만 채택 |

## 응용 과제

1. 본인 제품 이미지 1장 → 같은 톤 캐러셀 5장 일괄 (`generate_image` × 5)
2. 본인 채널 인플루언서 컨셉 영상 5초 → 바이럴 점수 80+ 목표로 3번 이내 다듬기
3. Part 5 콘텐츠·카피 에이전트와 결합 → 이미지·영상에 광고 카피 자동 합성
