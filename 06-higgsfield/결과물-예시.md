## 🧠 마케터 활용법 6가지

### A. SNS 광고 이미지 일괄 생성
```
"여름 캠페인 광고용 이미지 10장 변형으로"
→ generate_image × 10 (배경·앵글 변경)
```

### B. 제품 컷 영상화
```
"이 정적 제품 이미지를 회전 영상 5초로"
→ generate_video (image-to-video)
```

### C. 캐릭터 일관성 유지 (Personal Clipper)
```
"본인 사진 학습 → 캐릭터 그대로 50개 다양한 상황"
→ personal_clipper_create + 일관 생성
```

### D. 광고 후크 영상 자동
```
"3초 후크 영상 5패턴: 시각적 임팩트 강한 순간"
→ generate_video 5회 + 톤 변형
```

### E. 인플루언서 협업 시뮬레이션
```
"본인 브랜드 인플루언서 협업 예상 사진 (인플루언서 본인 사진 학습)"
→ personal_clipper + 협업 시나리오
```

### F. 바이럴 예측
```
"이 영상의 인스타·틱톡 바이럴 가능성 점수 + 개선안"
→ virality_predictor
```

## 💡 자주 묻는 질문

**Q. 한국 모델 가능?**  
A. 네. "20대 한국 여성/남성" 프롬프트로 자동 인식. 학습 모델에 다양한 인종 포함.

**Q. 본인 얼굴 학습 (Personal Clipper)?**  
A. 가능. 본인 사진 10~20장 업로드 → 학습 완료 후 캐릭터 일관성 유지된 50+ 이미지.

**Q. 영상 길이는 얼마까지?**  
A. 5초·10초·30초 옵션. 더 긴 영상은 5초 클립 여러 개 + Hyperframes(2-7)로 합성.

**Q. 저작권 안전?**  
A. Higgsfield 라이선스 상 상업 사용 OK. 단 실존 인물·캐릭터 모방 시 별도 권리 확인.

**Q. 무료 크레딧?**  
A. 가입 시 100 크레딧 + 매월 갱신. 본격 사용은 Pro 플랜 권장.
