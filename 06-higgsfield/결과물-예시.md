# 클립 2-3 결과물 예시 · Marketor Nourishing Face Cream 케이스 스터디

> 화장품 신제품 광고 자료를 Claude + Higgsfield 로 만든 실제 시연 흐름.
> **모델 이미지 1장 + 8초 광고 영상 1편** · 총 약 16~22 크레딧 · 약 5분.

## 한 줄 요약

제품 사진 1장 + 컨셉 한 줄 → Claude 가 포토 디렉터로 4단계 (기획 → 크레딧·모델 안내 → 생성 → 수정) 자동 진행 → 모델 키비주얼 1장 + 8초 광고 영상 1편.

---

## 🎬 시연 핵심 흐름 (실제 진행 로그 기반)

### Step 0 · 헬스 체크 (Claude 자동, 약 5초)

```
mcp__higgsfield__balance     → {credits: 203.52, plan: starter}
mcp__higgsfield__list_workspaces → private 워크스페이스 1개
```

Claude 응답 :
```
✅ Higgsfield 연결 정상
   · 잔여 크레딧 : 203.52 (Starter)
   · 이미지 1장 ≈ 1~3 / 영상 5~10초 ≈ 10~30
   · 약 이미지 70장 또는 영상 7~10편 가능
```

### Step 1 · 콘텐츠 유형 선택 (Gate 1)

```
🚪 화장품 콘텐츠 5가지 중 선택

A 🛍 제품 광고 1편 (URL → 9:16 광고)     · 9:16 8초  · ~20 크레딧
B 📸 제품샷 멀티컷 (4컷 A/B 테스트)        · 1:1 5초×4 · ~60 크레딧
C 🎥 무드컷 시네마틱 (텍스처·고급감)       · 9:16 8초  · ~30 크레딧
D 👩 UGC 리뷰 스타일 (아바타 사용)         · 9:16 10초 · ~40 크레딧
E ✂️ 유튜브 리뷰 → 숏폼 10개              · 9:16 ×10 · ~50 크레딧
```

→ 사용자가 **모델 + 제품 광고** 선택 → 본 시연 진입

### Step 2 · 컨셉 픽 (Gate 2 · 모델 사진 컨셉 3종)

같은 톤(클린·자연·아침) 유지하면서 모델만 다르게.

| # | 컨셉 | 분위기 | 추천 모델(MCP) | 활용처 |
|---|---|---|---|---|
| **M1 ★** | 에디토리얼 클로즈업 | 잡지 화보 · 프리미엄 | `soul_2` | 브랜드 키비주얼·상세페이지 메인 |
| M2 | 모닝 루틴 라이프스타일 | 햇살 욕실·창가 · 일상감 | `marketing_studio_image` | 인스타 피드·콘텐츠 |
| M3 | UGC 발색 리뷰 | 셀카 톤 · 진정성 | `marketing_studio_image` + 아바타 | 틱톡·릴스 후크 |

→ Claude 추천 : **M1** (Marketor 의 클린·미니멀 톤에 가장 적합, 영상 시작 프레임 연결 용이)

### Step 3 · Claude 디렉팅 (Gate 3 · 프롬프트 확정)

사용자 한 줄 : *"M1 에디토리얼 + 20대 후반 한국인 여성 + 글로우 클린 피부 + 9:16"*

Claude 가 자동으로 **시각 디렉션** 10항목 보강 :

| 항목 | Claude 디렉팅 |
|---|---|
| 각도 | 3/4 측면, 카메라 쪽으로 살짝 |
| 표정 | 차분·고요, 눈 반쯤 (아침 명상 톤) |
| 포즈 | 왼손이 잔을 쇄골 근처에서 부드럽게 받쳐 듦 |
| 의상 | 오프화이트 오버사이즈 린넨 셔츠 |
| 헤어 | 어깨 길이 자연 웨이브, 사이드 파트 |
| 메이크업 | 미니멀, 글로시한 연한 틴트 |
| 배경 | 아침 햇살 인테리어 + 우측 유칼립투스 보케 |
| 컬러 톤 | 웜 포셀린 피부 · sage · cream · soft white · dusty navy |
| 라이팅 | 좌상단 45° 키라이트, 5200K, 부드러운 랩어라운드 |
| 카피 공간 | 좌상단 1/3 네거티브 스페이스 |

→ 한국어 한 줄 → 영문 완성 프롬프트 약 25줄 자동 생성

### Step 4 · 크레딧·모델 안내 (Gate 4)

```
💰 사용 예정 안내
  · 모델 : soul_2 (Higgsfield 에디토리얼 포트레이트 최적)
  · 예상 크레딧 : 약 0.24 (2장, 2k 기본)
  · 잔여 크레딧 : 203.52 → 203.28 (예상)
  · 진행할까요? (y/n)
```

→ 사용자 `y` 확인

### Step 5 · 이미지 생성 (모델 키비주얼 2장 · 약 30초)

```
mcp__higgsfield__generate_image
  prompt: <Step 3 완성 프롬프트>
  model: soul_2
  aspect_ratio: 9:16
  count: 2
```

결과 :
```
✅ 모델 이미지 2장 완료
  · Job 1 · 3ceab6b7 · CloudFront URL
  · Job 2 · 768842cb · CloudFront URL
  · 크레딧 차감 : 0.24 (잔액 203.28)
```

→ 사용자가 마음에 드는 1장 선택 → 영상 `start_image` 로 체이닝

### Step 6 · 영상 생성 (8초 광고 영상 · 약 2~3분)

선택한 모델 이미지를 시작 프레임으로 사용. 8초 스토리보드 :

| 시간 | 액션 | 카메라 |
|---|---|---|
| 0~2s | 잔잔한 아침 빛, 모델 표정 변화 없음 | 느린 푸시-인 (4~5°) |
| 2~4s | 흰 뚜껑이 천천히 살짝 들리며 옆으로 기울어짐 | 푸시-인 계속, 초점 라벨→내용물 |
| 4~6s | 잔 안 크림 표면, 나무 스푼 위 크림 한 방울 윤기 | 살짝 내려다보는 미세 틸트 |
| 6~8s | 햇살 플레어 좌측에서 부드럽게 번지며 마무리 | 정지, 부드러운 sun flare |

```
mcp__higgsfield__generate_video
  prompt: <Claude 가 기획한 8초 영상 디렉팅>
  model: seedance_2_0
  aspect_ratio: 9:16
  duration: 8
  resolution: 720p
  genre: drama
  medias: [{role: start_image, value: <Job 1 URL>}]
```

결과 :
```
✅ 영상 생성 완료
  · 길이 8초 · 9:16 · 720p
  · 모델 : Seedance 2.0 (Bytedance · reference-driven, 제품 일관성 최강)
  · 크레딧 차감 : 약 15~20 (잔액 약 183)

📊 바이럴 점수 예측 (옵션) : 74/100
```

## 🎯 시연에서 배우는 5가지 인사이트

### 1. 모델 사진 먼저, 영상은 그 이미지를 시작 프레임으로 체이닝
영상에 인물이 들어가면 이미지부터 만들어 톤·구도를 확정한 다음 영상으로 연결. 영상은 크레딧이 크니까 한 번에 잘 나와야 함.

### 2. 컨셉 옵션 3종 → 모델 프로필 → 비율 → 카피 공간 (다층 게이트)
"광고 만들어줘" 한 줄로 끝내지 않고 단계별 확정. Claude 가 추천하면 사용자는 yes/no 만. 사고가 막히지 않음.

### 3. 사진이 인라인 첨부면 파일 경로 없음 → 라벨·디테일 묘사로 우회
제품 사진을 채팅에 올려도 파일 경로가 없으면 `media_upload` 불가. Claude 가 라벨 텍스트·재질·색상을 영문 프롬프트로 정확히 묘사해 동등한 결과 도출.

### 4. 비용 프리플라이트 (`get_cost: true`) 가 예산 통제의 핵심
호출 전 정확한 크레딧 (0.24 같이 소수점까지) 표시. 잔여와 비교 후 진행 결정. 영상은 크니까 더 중요.

### 5. Claude 디렉팅이 크레딧 절감의 본질
한 줄 ("모델 사진") → Claude 가 10항목 (각도·표정·포즈·의상·헤어·메이크업·배경·컬러·라이팅·카피 공간) 자동 보강 → 한 번에 원하는 결과 → 재생성 0 → 크레딧 절약.
