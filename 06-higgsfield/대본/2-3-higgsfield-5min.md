## 촬영 대본

### [0:00~0:25] Cover

〔슬라이드: 01〕

> "Part 2 여섯 번째 도구는 Higgsfield(힉스필드) MCP입니다. **광고용 이미지·영상·장면 전환용 짧은 영상을 한 줄 명령으로 즉시 생성**해주는 도구예요.
>
> 본 클립이 끝나면 디자이너·외주 없이 광고 이미지 1장 30초, 5초 영상 1~3분에 만들 수 있습니다."

### [0:25~1:30] Before/After · 외주 1일 → 1~3분

〔슬라이드: 02〕

> "광고 캠페인마다 비주얼 자산 (이미지·영상·모델컷) 이 매번 필요합니다. 현재 옵션이 3가지인데 모두 비효율적이에요.
>
> 첫째 **광고 모델 촬영 1회** · 모델 섭외비 200~500만원 + 스튜디오 대여 30~50만원 + 헤어·메이크업 20~40만원 + 사진작가·영상감독 50~200만원 + 의상·스타일링 20~50만원. **합쳐서 한 번에 약 300만원~800만원.** 인기 모델·셀럽이면 1000만원+. 게다가 결과물 받기까지 1~2주 대기.
>
> 둘째 **디자이너 일러스트·이미지 외주** · 1장 5~10만원 + 며칠 대기. 모델 컷은 없고 제품·일러스트 위주.
>
> 셋째 **이미지 편집 프로그램 직접 작업** · 한 장 만드는 데 30분~2시간. 디자인 툴 사용법을 따로 배워야 하고, 모델 사진은 사실상 만들 수 없어요.
>
> **Before(전)** · 광고 1편 (모델 + 영상) = 300만원~1000만원 + 1~2주 대기.
>
> **After(후)** · Claude(클로드) 가 4단계로 자동 진행. 메인 이미지 1장 (약 0.24 크레딧) → 그 이미지를 시작 프레임으로 8초 영상 (약 15~20 크레딧). **합쳐 약 16~22 크레딧 (한화 약 5천원~1만원) + 5분.**
>
> 약 300배 절감, 시간으로는 2주 → 5분. 그것도 **인물 일관성 100%** 유지하면서. 어도비·미드저니 같은 외부 도구를 따로 배우거나 결제하지 않아도 됩니다."

### [1:30~3:00] 5 시나리오

〔슬라이드: 03〕

> "Higgsfield 로 만드는 광고 비주얼 5가지입니다. 본 강의 시연은 A 이미지 → B 영상 이어가기 흐름으로 진행합니다.
>
> **A. 메인 이미지 (모델 컷) ★** · '20대 후반 한국 여성 + 클린 글로우 피부 + 화장품 잔을 받쳐 든 에디토리얼 컷' 30초~1분. **본 강의 시연 1.** 인물이 들어가는 콘텐츠는 무조건 여기부터.
>
> **B. 모델 이미지 → 8초 광고 영상 (이어가기) ★** · 실습 1 의 모델 이미지를 영상의 시작 프레임으로 그대로 사용. '잔 뚜껑이 천천히 열리고 햇살 플레어로 마무리' 약 2~3분. **본 강의 시연 2.** 인물·제품 외관 100% 유지.
>
> **C. 인스타 캐러셀 5장 일괄** · '같은 톤으로 5장' 3~5분. 피드 운영용.
>
> **D. 장면 전환용 짧은 영상** · '신상품 클로즈업 회전 5초' 1~3분. 영상 편집에 끼워 넣는 보조 클립 (업계 용어 B-roll).
>
> **E. 바이럴 점수 예측** · '이 영상 바이럴 가능성과 개선점' 30초. AI가 영상 보고 점수 예측.
>
> 시작 순서가 중요해요. **인물·영상 콘텐츠는 A 이미지 → B 영상 이어가기가 정답입니다.** 영상부터 만들면 인물·제품 일관성이 깨지고 재생성 → 크레딧 낭비. 이미지부터 톤 확정 → 그 이미지를 영상 시작 프레임으로 → 한 번에 완성."

### 🎬 [3:00~4:00] 작동 원리 · 한 줄 명령이 광고 이미지 1장이 되기까지

〔슬라이드: 04 작동 흐름 · "higgsfield 시작하자" → 메뉴 → 번호 선택 → 자동 실행〕

**🎙️ 강사 발화:**

> "실제로 어떻게 작동하는지 원리를 짚어드릴게요. 설치가 끝난 상태라면, Claude한테 단어 하나만 입력하면 됩니다.
>
> **`higgsfield 시작하자`** 라고 입력하세요. 슬래시 명령 아니에요. 본 강의가 미리 만들어둔 메뉴 화면이 자동으로 펼쳐집니다.
>
> 그러면 Higgsfield(힉스필드) 가 설치된 걸 자동 감지해서 **작업 가능 업무 5개 메뉴**를 보여줍니다 · 광고 이미지·5초 오프닝 영상·인스타 5장 일괄·장면 전환용 짧은 영상·바이럴 가능성 예측. 본인은 '**1**' 만 누르면 끝이에요.
>
> 여기서 가장 중요한 부분이 있습니다. **본인이 던진 한 줄을 Claude 가 그대로 힉스필드에 넘기지 않습니다.** Claude 가 먼저 포토 디렉터 역할로 프롬프트를 기획해줍니다 · 카메라 앵글·조명·구도·색감·제품 배치·카피 위치를 다듬어서 완성된 프롬프트로 변환. 그 다음에 힉스필드를 호출합니다.
>
> 호출 직전에 **두 가지를 더 알려줍니다.** ① 이 작업이 크레딧을 얼마나 소진할지 미리 표시. 광고 이미지 1장이면 약 1~2 크레딧, 5초 영상이면 약 5~10 크레딧. 잔여 크레딧과 비교해서 "지금 진행해도 OK" 또는 "이번 달 크레딧 한도 임박" 같이 안내. ② 어떤 모델을 쓸지 추천. 광고 이미지엔 모델 A, 시네마틱 영상엔 모델 B, 인물·제품 컷엔 모델 C. **본인이 모델 이름을 외울 필요가 없어요. Claude 가 알아서 골라 알려줍니다.**
>
> 여기서 중요한 점 한 가지 · **크레딧은 한정된 자원입니다.** 무료 50, 유료도 한도가 있어요. 잘못된 한 줄을 던지면 결과가 부족해서 다시 생성 → 크레딧 낭비 → 캠페인 1건 못 끝내고 끝납니다. 그래서 **사전에 Claude 와 기획을 잘 잡는 게 크레딧 절감의 핵심**이에요. 한 번에 원하는 결과를 뽑아내는 디렉팅 능력 = 마케터의 새 역량입니다. 클로드코드를 잘 다루는 게 곧 광고 비주얼 예산 관리입니다.
>
> 미드저니나 달리에 마케터가 직접 한 줄 던지면 결과가 들쭉날쭉합니다. 같은 명령 두 번이면 두 가지 다른 결과. 본인이 직접 카메라 용어·조명 용어를 알아야 일정한 퀄리티가 나와요. 하지만 본 강의 흐름은 **Claude 가 디렉터로 프롬프트를 기획하고 힉스필드는 그대로 그리기만 합니다.** 별도 포토 디렉터 한 명이 옆에 있는 것과 같은 효과입니다.
>
> 그래서 약 30초~1분 안에 메인 이미지 (모델 컷) 1장이 완성됩니다. 약 0.24 크레딧 소진. 마음에 안 들면 한 줄 더 던지세요 · '색감을 더 따뜻하게'·'제품을 가운데로'. Claude 가 다시 디렉팅하고 힉스필드가 다시 그립니다.
>
> 그 다음이 본 흐름의 핵심입니다 · **이미지가 OK 면 그 결과 이미지를 영상의 시작 프레임으로 그대로 이어갑니다.** Claude 가 8초 스토리보드를 자동 기획 (0~2초 후크 / 2~4초 본론 / 4~6초 클라이맥스 / 6~8초 마무리) → seedance 영상 모델이 그 이미지에서 출발해 **인물·제품·톤을 100% 유지하면서** 8초 영상을 만들어냅니다.
>
> 이 "이미지 → 영상 이어가기" 가 곧 **비용 절감의 결정적 한 수**입니다. 영상은 한 편에 15~20 크레딧. 영상부터 만들면 인물이 매번 다르게 나와 재생성 → 한 번 재생성에 30~40 크레딧이 사라집니다. 반면 이미지부터 0.24 크레딧으로 톤·인물·제품 확정 → 그 이미지를 시작 프레임으로 영상 한 번에 완성. **이미지 → 영상 이어가기 = 인물 일관성 100% 유지 + 재생성 0 + 크레딧 절감**.
>
> 실전 노하우 4가지를 짧게 짚어드릴게요.
>
> **첫째, 제품 라벨·로고가 있는 컷은 '글자 잘 쓰는 모델' 을 써야 합니다.** 일반 인물 모델은 라벨 글자를 흐릿하게 만들거나 가짜 글자로 채워 넣어요. 힉스필드 안에는 모델 종류가 여러 가지인데, 라벨·텍스트가 들어가면 텍스트 전용 모델 (예: nano_banana_pro), 인물 사진은 인물 전용 모델 (예: soul_2) 이 따로 있습니다. **Claude 가 자동으로 라벨 유무를 감지해서 적합한 모델을 골라줍니다.** 본인은 모델 이름을 외울 필요 없어요.
>
> **둘째, 호출 전 정확한 크레딧 견적 확인 필수.** 추정 15~20 크레딧이 실제 호출하면 36 크레딧 (약 2배) 나오는 경우 흔합니다. Claude 가 호출 전에 '이번 작업은 약 X 크레딧 소진' 미리 알려주니까 견적 보고 진행 결정.
>
> **셋째, 어색한 빛 효과는 빼라고 미리 알려줘야 합니다.** 영상에서 햇살이 얼굴에 번지는 효과 (sun flare·렌즈 플레어·반짝임) 가 자연스럽게 나올 줄 알지만, 모델이 과하게 적용해서 얼굴에 어색한 빛줄기가 깔리는 경우가 많아요. 자연 채광만 원하면 Claude 한테 '햇살 효과는 빼고 자연 채광만' 라고 명시해주면 됩니다.
>
> **넷째, 힉스필드 무료 요금제는 일부 프리미엄 영상 모델이 막혀 있습니다.** Claude 가 자동으로 무료에서 쓸 수 있는 다른 모델로 갈아타니까 본인은 그냥 진행하시면 돼요. 결과 퀄리티는 거의 차이 없습니다.
>
> 결과물은 1080×1080 광고 이미지 1장입니다 · 색감 + 제품 + 카피가 한 장에 일관되게 담깁니다. (실제 실행 화면은 별도 실습 영상에서 보여드립니다.)
>
> 무료 크레딧 한도 안에서 매일 1~2장 가능. 캠페인 한 건당 5~10장이 필요하면 그것도 한 번에 묶어서 요청할 수 있어요."

### [4:00~4:35] 마케터 가치 3가지

〔슬라이드: 05〕

> "Higgsfield 의 마케터 가치 3가지.
>
> **첫째, 광고 외주비·툴 학습 0원** · 1장 5~10만원 외주비가 사라지고 (월 30~50만원 절감), 디자인 프로그램이나 다른 AI 도구를 따로 배우거나 결제할 필요도 없어요.
>
> **둘째, 포토 디렉터 효과** · Claude 가 먼저 프롬프트를 기획·디렉팅해주고 힉스필드가 그대로 그립니다. 본인이 카메라 용어·조명 용어를 몰라도 일정한 퀄리티가 보장됩니다. **호출 전에 크레딧 소진량과 적합한 모델까지 미리 알려주니까** 예산 관리도 같이 됩니다. 별도 포토 디렉터 한 명을 옆에 두는 것과 같은 효과예요.
>
> **셋째, 한 줄로 즉시 다시 만들기** · 결과가 마음에 안 들면 '색감 더 따뜻하게'·'제품 위치 가운데로' 한 줄 추가로 30초 안에 다시. Claude 가 다시 디렉팅하고 힉스필드가 다시 그립니다.
>
> 한 줄 정리 · **포토 디렉터 + 일러스트레이터 한 팀이 본인 옆에 생긴다.**"

### 🎬 [4:35~5:00] Closing · 설치는 Part 2 마지막 영상에서

〔슬라이드: 06 Sunset + Higgsfield 실제 시연 영상 안내〕

**🎙️ 강사 발화:**

> "Higgsfield 의 개념과 작동 원리는 여기까지입니다.
>
> 비용 한 가지 짚어드리면 · **힉스필드 무료 요금제 (가입 즉시 50 크레딧 제공)** 으로 본 강의 시연 (메인 이미지 1장 + 8초 영상 1편 = 약 16~22 크레딧) 충분히 가능합니다. 일부 프리미엄 영상 모델은 유료 요금제 가입이 필요하지만 Claude 가 자동으로 무료에서 쓸 수 있는 모델로 갈아타니까 본인은 그냥 진행하시면 돼요.
>
> 설치는 따로 진행해요. 본 강의 Part 2 **마지막 영상**이 '설치 마라톤' 영상입니다. 11개 MCP 이론을 다 보신 후, 마지막 영상에서 Claude Code에 '**MCP 전체 설치하자**' 한 줄 입력하면 11개 MCP가 한 번에 자동 설치됩니다. 사전 준비물 같은 자세한 안내도 그 영상에서 함께 다룹니다.
>
