## Phase 1 · YouTube Data API v3 (공개 데이터)

### Step 1. 두 API 활성화 (Google Cloud Console)

```
GCP Console → APIs & Services → Library
  ① "YouTube Data API v3"     → Enable   (필수 · Phase 1)
  ② "YouTube Analytics API"   → Enable   (권장 · Phase 2 에서 사용)
```

> 💡 두 API 모두 같은 프로젝트에서 활성화. Phase 2 에서 다시 안 들어가도 됨.

### Step 2. API Key 발급

```
APIs & Services → Credentials → + Create Credentials → API Key
  → 키 복사 (AIza... 39자)
```

### Step 3. .env 등록

```bash
# marketing-os/.env
YOUTUBE_API_KEY=AIza...
```

### Step 4. .mcp.json 등록

```json
{
  "mcpServers": {
    "youtube-data": {
      "command": "npx",
      "args": ["-y", "youtube-data-mcp-server"],
      "env": { "YOUTUBE_API_KEY": "${YOUTUBE_API_KEY}" }
    }
  }
}
```

### Step 5. 헬스 체크 (본인 채널이 없어도 OK)

```bash
claude "유튜브에서 '클로드코드 마케팅' 키워드로 검색한 영상 TOP 10을 조회수 순으로 정리"
```

## Phase 1 시연 · 마케터 5패턴 (Data API v3)

```
[역할] D2C 브랜드의 유튜브 콘텐츠 마케터.

[입력] 본인 채널 ID (UCxxxxxx) 최근 30일.

[산출물]
  1. 영상별 (제목 · 업로드일 · 조회 · 좋아요 · 댓글) 표
  2. 평균 대비 1.5배 이상 잘된 영상 TOP 3 + 공통 패턴
  3. 댓글 100개 샘플 → 긍정/부정/질문 분류

[제약] 한국어. 통계는 정수 + 콤마.

[검증] 데이터 추출 기간과 영상 수를 명시.
```

### 그 외 Phase 1 시나리오

| 시나리오 | 한 줄 명령 | 소요 |
|---|---|---|
| 경쟁사 채널 분석 | "경쟁사 채널 UC... 최근 20개 영상 성과 표로" | 30초 |
| 키워드 트렌드 | "'AI 마케팅' 키워드 영상 TOP 30 분석 + 공통 후크" | 1분 |
| 댓글 카테고리 분류 | "이 영상 댓글 100개 → 긍정/부정/질문 + 우선 답변 5개" | 1분 |
| 영상 후크 추출 | "이 채널 베스트 영상 5개의 첫 15초 카피 추출" | 30초 |
| 시리즈 기획 | "이 채널 인기 영상 패턴 → 다음 5개 영상 기획안" | 1~2분 |

## Phase 2 · YouTube Analytics API (본인 채널 운영 분석)

### 사전 셋업

→ [`analytics-api-oauth.md`](analytics-api-oauth.md) 의 STEP 1~6 진행 (약 15분 · 한 번만)

설치 끝나면 `.env` 에 다음 4개가 모두 채워짐 :

```bash
YOUTUBE_API_KEY=AIza...                      # Phase 1
YOUTUBE_OAUTH_CLIENT_ID=...                  # Phase 2
YOUTUBE_OAUTH_CLIENT_SECRET=...              # Phase 2
YOUTUBE_REFRESH_TOKEN=...                    # Phase 2 (--auth 실행 후 자동)
```

### Phase 2 시연 · 본인 채널 운영 분석 5패턴

| 시나리오 | 한 줄 명령 | 산출 |
|---|---|---|
| **주간 구독 변동** | "내 채널 지난 7일 구독자 증감 + 일별 그래프" | `subscribersGained` / `subscribersLost` 시계열 |
| **이탈 영상 TOP 5** | "내 채널 최근 30일 구독 취소가 가장 많은 영상 TOP 5" | 영상별 `subscribersLost` 순 + 가설 |
| **트래픽 소스 분석** | "내 채널 최근 30일 시청 유입 경로 비율 + 변화" | 검색·외부·추천 영상 비율 |
| **시청자 인구통계** | "내 채널 시청자 성별·연령·지역 분포" | `viewerPercentage` 도표 |
| **시청 지속률** | "내 채널 영상 평균 시청 지속률 + 구간별 이탈 지점" | `audienceWatchRatio` |

### Phase 2 시연 명령 예시

```
[역할] 본인 채널 운영 분석가.

[입력] 본인 채널 (Analytics API 자동 인식 · 채널 ID 입력 불필요)
기간 : 지난 30일

[산출물]
  1. 주간 구독 증감 그래프 (Gained vs Lost)
  2. 구독 취소 TOP 5 영상 + 가설 (영상 길이·주제·업로드 시점)
  3. 트래픽 소스 5종 비율 + 전 주 대비 변화
  4. 시청자 인구통계 (성별·연령·지역 TOP 3)
  5. 개선 액션 3가지 (가장 큰 이슈 우선)

[제약] 한국어. 수치는 정수 + 콤마.
```

## 트러블슈팅

| 증상 | Phase | 원인 | 해결 |
|---|---|---|---|
| `quotaExceeded` | 1 | 일일 10,000 유닛 초과 | 새벽 12시 리셋 또는 quota 증액 신청 |
| `403 forbidden` (댓글 쓰기) | 1 | API Key 만으로는 불가 | OAuth 로 전환 (별도) |
| `access_denied` | 2 | OAuth 테스트 사용자 미등록 | analytics-api-oauth.md STEP 2-5 |
| `invalid_grant` | 2 | refresh_token 만료 | `.env` 의 `YOUTUBE_REFRESH_TOKEN=` 비우고 `--auth` 재실행 |
| `403 channelNotMonetized` 류 | 2 | Analytics API 비활성화 | analytics-api-oauth.md STEP 1 재확인 |
