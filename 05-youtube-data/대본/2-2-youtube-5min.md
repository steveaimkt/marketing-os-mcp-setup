### 🎬 [0:25~1:30] Before/After · 공개 데이터 + 본인 채널 운영 분석

〔슬라이드: 02 Phase 1 (공개 데이터) + Phase 2 (운영 분석) 통합 비교〕

**🎙️ 강사 발화:**

> "유튜브 마케터가 매주 반복하는 작업을 두 묶음으로 정리하겠습니다.
>
> **첫 번째 묶음 (Phase 1 · 공개 데이터)** · 영상별 성과 정리 1시간 + 새 댓글 답글 1~2시간 + 트렌드 키워드 검색 30분 + 경쟁 채널 모니터링 30분. 합쳐 **주 3시간**. Data API v3 + API Key 만 있으면 끝.
>
> **두 번째 묶음 (Phase 2 · 본인 채널 운영 분석)** · 본인 채널의 구독 변동·이탈 원인·트래픽 소스·시청자 인구통계 분석. 지금까지는 YouTube Studio 들어가서 일일이 확인하셨을 거예요. 매주 1시간씩. 자동화 시도하면 막힙니다. 공개 데이터가 아니라 **채널 소유자만 볼 수 있는 데이터**여서 API Key 만으로는 불가능합니다.
>
> **Before(전)** · Phase 1 주 3시간 + Phase 2 매주 1시간 = 거의 **주 4시간**이 통계·분석에 사라집니다.
>
> **After(후)** · Claude(클로드) 에 한 줄. 'Phase 1 영상 성과 정리해줘' → 30초. '내 채널 지난 7일 구독 변동과 이탈 영상 TOP 5 분석해줘' → 1분. 두 묶음 통합 약 5분.
>
> **약 48배 단축.** 그 시간을 콘텐츠 기획에 투입할 수 있고, 무엇보다 **이전엔 거의 불가능했던 구독 이탈 원인 자동 분석이 가능해진다**는 게 핵심입니다."

### 🎬 [1:30~3:00] 5 시나리오 · 유튜브 마케터 5가지 자동화

〔슬라이드: 03 5 시나리오 카드〕

**🎙️ 강사 발화:**

> "유튜브 마케터 작업 5가지가 어떻게 자동화되는지 보겠습니다.
>
> **A. 영상별 성과 정리 ★** · '우리 채널 최근 20개 영상 조회·좋아요·댓글 표로' 30초. 매주 월요일 첫 번째 표.
>
> **B. 댓글 답글 자동 분류 ★** · '새 댓글 50개를 긍정·질문·불만으로 분류, 답글 초안 작성' 2~3분. **수동 1~2시간이 몇 분으로.**
>
> **C. 트렌드 키워드 검색** · 'K뷰티 관련 최근 1개월 인기 영상 TOP 20' 1분. 다음 콘텐츠 기획에.
>
> **D. 경쟁 채널 모니터링** · '경쟁 채널 3곳 최근 영상·평균 조회수 비교' 1분.
>
> **E. 본인 채널 구독 변동·이탈 분석 ★ (Phase 2)** · '내 채널 지난 7일 구독자 증감 + 이탈 영상 TOP 5'. Analytics API + OAuth 별도 셋업 (15분 한 번). 본인 채널만 가능한 깊이 있는 운영 분석.
>
> 시작 순서 · 처음이면 A부터 (가장 자주). 익숙해지면 B 댓글 분류 (시간 절약 최대). 본인 채널 운영자라면 E Phase 2 OAuth 셋업 (구독 이탈 원인 분석)."

〔→ 04 작동 원리〕

### 🎬 [3:00~4:00] 작동 원리 · 한 줄 입력 → 메뉴 7개 → 자동 실행

〔슬라이드: 04 작동 흐름 · "youtube-data 시작하자" → 메뉴 7개 (Phase 1·2) → 번호 선택 → 자동〕

**🎙️ 강사 발화:**

> "실제로 어떻게 작동하는지 쉽게 풀어드릴게요. 설치만 끝났다면 Claude 한테 평문 한 문장 던지면 됩니다.
>
> **`youtube-data 시작하자`** 라고 입력하세요. 슬래시 명령 아니에요. 본 강의가 미리 만들어둔 메뉴 화면이 자동으로 펼쳐집니다.
>
> 메뉴는 **두 묶음으로 나뉘어 보여요.**
>
> **Phase 1 (메뉴 1~4) · 공개 데이터** · 영상 성과 정리·댓글 분류·키워드 트렌드·경쟁 채널 모니터링. API Key 만 있으면 바로 작동합니다.
>
> **Phase 2 (메뉴 5~7) · 본인 채널 운영 분석** · 구독 변동·이탈 영상 TOP 5·트래픽 소스. **본인 채널 인증을 한 번 등록**해두면 자동으로 인식해서 메뉴가 활성화됩니다.
>
> 본인은 그냥 **번호만 누르면 끝**입니다. '1' 을 누르면 Phase 1 시연, '6' 을 누르면 본인 채널 구독 이탈 영상 분석.
>
> Claude 가 알아서 유튜브한테 필요한 데이터를 요청하고, 받은 결과를 표로 정리해줍니다. 약 30초~1분 안에 결과가 나옵니다. 어떤 데이터를 어떻게 가져올지는 신경 쓰지 않으셔도 돼요. Claude 가 알아서 골라 씁니다.
>
> 그리고 이게 1회성이 아닙니다. **Part 7 의 youtube-channel-analyzer 에이전트** 가 매주 같은 작업을 본인 대신 자동 실행하고 결과를 디스코드로 발송. 본인이 명령을 한 번도 입력하지 않아도 매주 종합 리포트가 도착합니다."

〔→ 05 마케터 가치〕

### 🎬 [4:00~4:35] 마케터 가치 3가지

〔슬라이드: 05 가치 3 카드〕

**🎙️ 강사 발화:**

> "YouTube Data MCP 의 마케터 가치 3가지.
>
> **첫째, 시간 절감** · 영상 성과 정리·댓글 답글·트렌드 검색·경쟁사 모니터링이 매주 3시간에서 5분으로. 1년 약 150시간이 본인에게 돌아옵니다. 그 시간을 다시 콘텐츠 기획에 쓸 수 있어요.
>
> **둘째, 인사이트 있는 운영** · 본인 채널의 구독 이탈 원인·트래픽 유입 경로·시청자 인구통계가 자동으로 분석됩니다. YouTube Studio 들어가서 화면 일일이 확인하던 작업이 자연어 한 줄로. **직감이 아닌 데이터로 채널을 운영**하게 됩니다.
>
> **셋째, 경쟁력 확보** · 경쟁 채널 신작·인기 키워드 트렌드를 매일 자동 추적. 어떤 영상을 만들지·어떤 키워드를 노릴지 근거 기반으로 결정. Part 7 에이전트가 매주 종합 리포트를 디스코드로 발송합니다.
>
> 한 줄로 정리하면 · **본인 옆에 유튜브 매니저 + 채널 분석가 + 시장 전문가가 동시에 생긴다.**"
