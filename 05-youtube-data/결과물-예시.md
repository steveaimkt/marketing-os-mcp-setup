## 🧠 마케터 활용법 6가지

### A. 트렌드 키워드 모니터링
```
"매주 'AI 마케팅' 키워드 상위 10개 변동 추적"
→ search 도구 + Part 4 trend-monitor 에이전트
```

### B. 경쟁 채널 분석
```
"마케팅TV 채널의 지난 30일 영상별 성과 비교"
→ getChannelStatistics + getVideoDetails
```

### C. 본인 채널 KPI 추적
```
"내 채널 지난 7일 영상별 조회수·시청률 표"
→ getMyChannel + getVideoStatistics
```

### D. SEO 키워드 발굴
```
"'AI 마케팅' 카테고리 상위 영상 제목 공통 키워드 추출"
→ search + Claude 자동 분석
```

### E. 콘텐츠 아이디어 발견
```
"이번 달 가장 좋아요 비율 높은 마케팅 영상 10개 + 주제"
→ search + Claude 아이디어 도출
```

### F. 인플루언서 발굴
```
"구독자 1만~10만, 시청률 50% 이상, 마케팅 카테고리 채널 추천"
→ search + getChannelStatistics 필터링
```

## 💡 자주 묻는 질문

**Q. 비공개·미등록 영상 데이터도?**  
A. 본인 채널은 OAuth 추가 시 가능. 공개 영상은 API key 만으로 OK.

**Q. 일일 한도 10,000 unit 어떻게 계산?**  
A. search 1회 = 100 unit, getVideo 1회 = 1 unit. 마케터 일상 사용엔 충분.

**Q. 한국어 검색 결과만 가능한가요?**  
A. `regionCode: 'KR'` + `relevanceLanguage: 'ko'` 자동 설정. 다른 국가도 명령에 명시.

**Q. 댓글 자동 분석도?**  
A. getCommentThreads 도구로 가능. Part 4 voc-analyzer 에이전트와 결합 권장.

**Q. 영상 자동 업로드?**  
A. 이 MCP는 읽기 전용. 업로드는 별도 API 권한 필요.
