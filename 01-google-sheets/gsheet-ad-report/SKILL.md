# Part 2 / 1-2 · Google Sheets 실습 — 광고 분석 리포트 자동 생성

> 광고 리포트 엑셀 1개를 던지면 Claude 가 집계·분석해 **구글 시트 분석 리포트**로 만든다.
> Google Sheets MCP 의 create/write/format/read 도구를 한 흐름에 시연하는 클립 1-2 실습 스킬.

## 전제 조건
- Google Sheets MCP 연결됨 (`mcp__google-sheets__*` 도구 노출). 안 보이면 `marketing-os` 루트에서 Claude Code 재시작.
- 분석할 광고 엑셀/CSV 가 `01-google-sheets/실습 데이터/` 에 있음 (기본). 없으면 사용자에게 경로 요청.
- `openpyxl` 필요: `pip3 install openpyxl --break-system-packages`

## 진행 포맷 (4단계 — 끊지 말고 한 흐름으로)

### 1️⃣ 안내 인사
다음 멘트로 시작:
```
📊 Google Sheets 실습을 시작합니다.
실제 광고 리포트 엑셀을 구글 시트 분석 리포트로 바꿔드릴게요.
Claude 가 데이터를 직접 집계하고, 무엇이 문제인지까지 짚어 색으로 강조합니다.
```

### 2️⃣ 실습 가능한 업무 (대표 5개 표로 안내)
| # | 업무 | 도구 |
|---|---|---|
| 1 | 분석 리포트 시트 자동 생성(다중 탭) | `create_spreadsheet` · `add_sheet` |
| 2 | 집계 데이터 기록 | `write_sheet` |
| 3 | 데이터 읽기·검증 | `read_sheet` · `batch_read_sheet` |
| 4 | 헤더·강조 서식 | `format_cells` |
| 5 | 구조 조회 | `get_spreadsheet_info` |
"오늘은 이 전부를 묶어 광고 리포트 1건을 만든다"고 안내.

### 3️⃣ 실습 결과물 (인터랙티브 — 자동 실행 금지)

⚠️ **절대 끝까지 자동 실행하지 말 것.** 아래 4개 게이트에서 매번 멈추고 사용자 답을 받은 뒤 다음으로 진행한다. 산출물 생성 도구(`create_spreadsheet`·`write_sheet` 등)는 게이트 ①②③ 통과 후에만 호출.

**게이트 ① 어떤 작업을 시작할까요?**
가능한 작업을 메뉴로 제시하고 선택받는다. 예:
```
어떤 분석을 만들까요?
 1) 키워드 성과 분석 (TOP 키워드·낭비 키워드)
 2) 일별 추세 리포트
 3) 지면·캠페인별 효율 비교
 4) 전체 종합 리포트 (1~3 + 인사이트)  ← 기본
```
→ 사용자 선택 대기.

**게이트 ② 파일/데이터 위치를 알려주세요.**
입력 엑셀/CSV 경로를 사용자에게 확인받는다(자동 탐색해 넘기지 말 것).
- 사용자가 경로를 주거나 "실습 데이터 폴더"라고 하면 그때 집계 실행:
  ```bash
  cd "<클립폴더>/gsheet-ad-report" && python3 analyze.py "<사용자가 준 경로 또는 생략>"
  ```
- 컬럼 자동 감지 결과 + 요약(광고비·매출·ROAS·키워드수·낭비)을 보고하고 다음 게이트로.

**게이트 ③ 어떤 결과물을 원하는지 알려주세요.**
산출물 형태·범위를 확인받는다: 시트 탭 구성(키워드성과/일별추세/분석), TOP 키워드 수, 제목 등.
→ 확정 후에야 실제 적재 시작.

  - `create_spreadsheet(title, sheet_titles=[...])`
  - `write_sheet` 로 각 탭 적재 (kw / day / 분석 섹션: KPI·area·camp·hero·waste + **데이터 기반 인사이트·액션 3개**, 수치 나열 금지)
  - `get_spreadsheet_info` → `format_cells` (헤더 `#1a73e8`+흰글씨+굵게+가운데, 낭비 섹션 `#fce4ec`)
  - `read_sheet` 로 1개 범위 재검증

**게이트 ④ 결과물을 보여주고 피드백 요청.**
완성된 시트 URL + 핵심 인사이트 요약을 제시하고 **멈춰서** 피드백을 받는다("이대로 좋을지 / 수정할 부분"). 승인되면 `<클립폴더>/{날짜}-실습결과물.md` 저장.

## 산출물 표준

## 트러블슈팅
| 증상 | 해결 |
|---|---|
| `mcp__google-sheets__*` 안 보임 | marketing-os 루트에서 Claude Code 재시작 후 `/mcp` 확인 |
| `No module named openpyxl` | `pip3 install openpyxl --break-system-packages` |
| 필수 컬럼 미감지 | analyze.py 의 RULES 에 해당 매체 헤더명 추가 |
| write 범위 초과 | TOP 키워드 수를 `--top` 으로 축소 |
