## PART A. 구글 시트 1page 리포트

### Step 1 — 데이터 주고 분석 요청 (발화 복사)

```
이 엑셀 광고 리포트를 분석해서 구글 시트에 1page 액션 리포트를 만들어줘.
파일: 실습 데이터/A01217152_pa_daily_keyword_20260101_20260131.xlsx
```

> Claude가 먼저 **무엇을 / 어디에 / 어떤 형태로**를 확인한다(인터랙티브 게이트). "광고 성과 / 시트 안 요약 표"로 답하면 진행.

### Step 2 — Claude가 자동 수행하는 것 (지켜보기)

```
① Python(pandas)으로 엑셀 집계   ← 1,521행을 직접 세지 않음
   · 전환매출·ROAS는 14일 기준 컬럼 사용 (쿠팡 표준)
   · 검색 vs 비검색 지면 분리
   · 매출 0 키워드 낭비 합 계산
② 새 스프레드시트 생성 → 6개 섹션 작성
   ① 핵심 KPI ② 캠페인별 ③ 지면별 ④ 매출 TOP ⑤ 낭비 키워드 ⑥ 액션 플랜
③ 서식: 숫자 우측정렬 · 판정 가운데 · ROAS/KPI값/낭비 볼드 · 섹션 색 밴드
```

생성된 시트 URL이 나오면 클릭해서 연다.

### Step 3 — 가독성 수동 마무리 (수강생 직접 · 2분)

> 현재 google-sheets MCP는 **열 너비·줄바꿈·행 높이**를 못 바꾼다. 이 마지막 손질만 시트에서 직접 한다.

```
① 열 너비 자동 맞춤
   A~F 머리글 전체 선택 → 경계선 더블클릭
   (또는 우클릭 → 열 크기 조정 → 데이터에 맞추기)

② 긴 텍스트 줄바꿈(Wrap) 켜기
   판정·액션이 들어간 열 선택 → 서식 → 줄바꿈 → "줄바꿈"

③ 행 높이 여유
   1~56행 선택 → 우클릭 → 행 크기 조정 → 25~28px
```

→ 이 3단계로 텍스트 잘림이 사라지고 한눈에 읽힌다.

## PART B. A4 인쇄용 HTML 리포트

### Step 1 — HTML 요청 (발화 복사)

```
같은 데이터로 A4 세로 1장에 들어가는 HTML 리포트도 만들어줘.
인쇄했을 때 텍스트가 페이지를 넘어가면 안 돼.
```

> Claude가 `html-report-template` 스킬로 `@page A4` + 자동 줄바꿈이 적용된 단일 HTML 파일을 만든다.

### Step 2 — 열어서 PDF로 저장

```
① 생성된 .html 파일을 더블클릭 → 브라우저로 열림
② ⌘+P (인쇄)
③ 대상: PDF로 저장 / 용지: A4 / 여백: 없음 / 배율: 100%
   → 정확히 1장으로 떨어짐
```
