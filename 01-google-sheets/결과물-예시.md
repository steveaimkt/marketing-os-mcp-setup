# 클립 1-2 결과물 예시 · 주간 종합 리포트

> Google Sheets MCP 설치 완료 직후 마케터가 받게 되는 첫 결과물 1개의 실제 모습. 본 문서로 "내가 받게 될 것이 무엇인지" 미리 확인한 뒤 본인 데이터로 그대로 따라하기.

## 한 줄 요약

사용자가 `/start` 한 줄 입력 → Claude 가 5단계 자동 실행 → 3~5분 후 **새 스프레드시트 1개 (4 시트 포함)** + Discord 알림 1건.


## 🧠 마케터 활용법 5가지

### A. 매주 월요일 09:00 자동화 (가장 가치 있음)
```
Part 10 (AX 협업) cron 트리거 등록:
  - 매주 월요일 09:00 → /start 자동 실행
  - 결과물 URL → Discord #marketing 채널
  - 사용자 명령 0회 · 자동 가동
연 환산 절감: 약 180시간 (주 3.5시간 × 52주)
```

### B. 캠페인 종료 직후 회고
```
사용자 입력: "어제 종료된 봄 신상 캠페인 회고 리포트 만들어줘"
→ Claude 가 캠페인 기간 데이터만 필터링해서 동일 5단계 실행
→ 새 스프레드시트 '봄 신상 캠페인 회고 2026-05'
```

### C. 월말 임원 보고
```
사용자 입력: "이번 달 종합 리포트 만들어줘 — 임원용으로 한 페이지 요약 포함"
→ Claude 가 한 달치 데이터 집계 + 임원용 1페이지 요약 시트 추가
→ 임원에게 URL 공유로 발송
```

### D. 신규 채널 도입 시 베이스라인
```
사용자 입력: "쇼피 채널 추가됐어 — 첫주 베이스라인 리포트 만들어줘"
→ Claude 가 쇼피 데이터 별도 시트 추가 + 기존 채널 대비 비교
```

### E. 본인 브랜드 색상으로 디자인 표준화
```
사용자 입력: "헤더는 본인 브랜드 색상 #BC4749 로 통일해줘"
→ Claude 가 format_cells 도구로 모든 시트 헤더 색상 변경
→ 한 번 적용 후 매주 자동 유지
```

## 💡 자주 묻는 질문

**Q. 본 리포트를 본인 브랜드 톤·언어로 바꿀 수 있나요?**  
A. 네. 한 번 "헤더는 #BC4749, 통화는 USD 로" 같이 요청하면 Claude 가 메모리에 저장해 매주 자동 적용.

**Q. 시트가 4개가 아니라 7개인데 다 분석되나요?**  
A. 네. Claude 가 `get_spreadsheet_info` 로 자동 시트 목록 파악 후 의미를 추론. 단, 시트 이름이 명확할수록 정확도 높음 (예: 'Sheet1' 보다 '매출_2026' 권장).

**Q. 영문 시트도 가능한가요?**  
A. 네. 시트 이름·헤더 모두 영문 OK. 다만 한글 시트 이름과 영문 혼용 시 일부 OS 에서 인코딩 깨질 수 있음 → 영문 통일 권장.

**Q. 매번 시트 ID 입력해야 하나요?**  
A. 첫 호출 후 `.env` 의 `GOOGLE_SHEETS_DEFAULT_ID` 에 자동 저장 옵션 제공. 이후 `/start` 한 줄로 즉시 실행.

**Q. Google Workspace 계정도 되나요?**  
A. 네. 단 OAuth 동의 화면에서 "내부" 사용자 타입 선택 가능 (테스트 사용자 추가 불필요).
