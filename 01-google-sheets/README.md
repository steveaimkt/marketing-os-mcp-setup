# 클립 1-2. Google Sheets MCP · 스프레드시트 업무 자동화

## 한 줄 요약

Google Sheets MCP(엠씨피)를 설치하면 Claude(클로드)가 본인 시트를 **읽고 쓰고 서식까지 자동으로 만든다**. 매출·광고·재고가 흩어진 시트 4개를 자연어 한 줄로 종합 리포트로 만든다.

## 마케터에게 왜 필요한가

- 마케팅 데이터는 결국 시트로 모임 · 매출(쿠팡·네이버·자사몰), 광고(Meta·Google·Naver·Kakao), 재고, 고객
- 수동 피벗·VLOOKUP(브이룩업) · SUMIF(섬이프) 수식 · 조건부 서식이 반복되며 **분석 1건당 15~30분**
- 종합 리포트 작성 1회 = **반나절(3~4시간)** · 매주 반복하면 분석 시간이 본업을 잠식
- Google Sheets MCP 설치 후 같은 작업이 **5~10분**. 매주 3시간 30분 절감, 연간 약 180시간

## 무엇이 가능해지나

| 케이스 | 자연어 명령 예시 | 시간 |
|---|---|---|
| A. 시트 탐색 | "스프레드시트에 어떤 시트가 있어?" | 5초 |
| B. 매출 분석 | "채널별 매출 비교해줘" | 30초 |
| C. ROAS(로아스) 분석 | "플랫폼별 ROAS 비교하고 예산 재배분 안 만들어줘" | 1~2분 |
| D. 재고 알림 | "안전재고 미달 상품 상태를 '긴급'으로 업데이트하고 빨간 배경" | 30초 |
| E. 자동 리포트 | "주간 리포트 스프레드시트 새로 만들고 종합 분석 정리" | 3~5분 |

## 10 MCP 도구 한눈

| 그룹 | 도구 | 기능 | 자연어 예시 |
|---|---|---|---|

| | `clear_sheet` | 범위 삭제 | "분석 시트 비우고 새로 기록" |
| **관리 4** | `add_sheet` | 새 시트 탭 생성 | "'채널분석' 시트 만들어줘" |

| | `format_cells` | 서식 적용 | "헤더 파란 배경 흰 글씨" |

## 사전 준비물

- Node.js(노드 제이에스) 18+ · 터미널에서 `node --version` 확인
- Google 계정 1개 (Google Cloud Console 접근용)
- 마케팅용 스프레드시트 1개 (없으면 빈 시트라도 OK)
- 5~10분의 OAuth(오어쓰) 설정 시간 (최초 1회만)

## 작동 방식

```
[사용자 자연어 명령]
   ↓
[Google Sheets MCP 서버 호출]
   ↓
① OAuth 2.0 인증 (한 번 등록, 토큰 자동 갱신)
   ↓
② Google Sheets API v4 호출 · 10개 도구 중 필요한 것
   ↓
③ 응답 받음 · 데이터 또는 작업 결과
   ↓
[Claude 가 분석·정리·요약]
   ↓
[사용자에게 자연어로 응답 + 시트 URL]
```

OAuth 토큰은 `mcp-server/token.json` 에 안전 저장 · 만료 시 자동 갱신.

## 트러블슈팅 미리보기

| 증상 | 원인 | 해결 |
|---|---|---|
| `액세스 차단됨` | OAuth 동의 화면에 테스트 사용자 미등록 | Google Cloud Console > OAuth 동의 화면 > 테스트 사용자에 본인 이메일 추가 |
| MCP 도구 안 보임 | `settings.json` 경로 오타 또는 Claude 재시작 안 함 | 절대경로 확인 (`cd mcp-server && pwd`) + Claude Code 완전 종료 후 재시작 |
| `Quota exceeded` | API 분당 60회 초과 | 60초 대기 후 재시도 |
| 토큰 만료 오류 | 60일 이상 미사용 | `rm mcp-server/token.json` 후 재시작 → 재인증 |
| `Permission denied` | 시트가 본인 소유가 아님 | 시트 공유 권한 확인 또는 본인 소유 시트로 시작 |

## 검증된 산출물

- 채널별 매출 분석 → 새 시트 자동 생성 + 헤더 파란 배경 서식
- 광고 ROAS 시뮬레이션 → 예산 재배분 시 추가 매출 ₩1,147만원 계산
- 재고 긴급 상품 식별 → 행 상태 자동 업데이트 + 빨간 배경 강조
- 주간 종합 리포트 → 4개 시트 새 스프레드시트 자동 생성 + URL 공유
