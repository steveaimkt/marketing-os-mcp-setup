# 클립 2-4 실습: 영상제작 트리오 설치 + 첫 영상

## 실습 한 줄 요약

`/mcp설치-영상제작` 스킬을 호출해 3 도구를 25~35분 안에 설치하고 53초 통합 영상 1개를 자동 생성한다.

## 사전 준비

본 클립 시작 전 다음 5가지를 확인하세요:

```
□ Node.js 22 이상       → 터미널에서 `node --version`
□ Chrome 설치           → 시스템에 일반적으로 있음
□ ffmpeg               → 본 실습 중 자동 설치 안내
□ uv 패키지 매니저       → 본 실습 중 자동 설치 안내
□ HeyGen 무료 계정      → https://app.heygen.com/signup
□ ElevenLabs 무료 계정  → https://elevenlabs.io 가입
```

API key는 본 실습 중간에 발급하므로 미리 안 받아도 됨.

## 단계별 실행

### Step 0. 스킬 호출

Claude Code 에 한 줄 입력:

```
영상제작 MCP 설치하자
```

자동 호출되는 스킬: `/mcp설치-영상제작`

이후 모든 단계가 스킬 안내에 따라 자동 진행됩니다. 사용자가 직접 명령을 입력하는 곳은 API key 두 개뿐.

### Step 1. Hyperframes 설치 (5~10분)

스킬 자동 진행:
- `marketing-os/hyperframes/` 폴더 생성
- `npx hyperframes@latest init .` 실행
- 15개 추가 스킬 설치 (`npx skills add heygen-com/hyperframes`)
- `npx hyperframes doctor` 환경 점검
- ffmpeg 없으면 `~/.local/bin/` 에 정적 바이너리 자동 설치

### Step 1 결과물: 6초 WeeklyKPI 영상

```
🎯 첫 결과물 만들어볼까요?

스킬: "ROAS 3.2x 들어가는 6초 KPI 영상 만들어줘"

→ Hyperframes 자동 작성 + 렌더
→ renders/weekly-kpi-{시각}.mp4 출력
→ open 명령으로 자동 재생
```

### Step 2. HeyGen MCP 설치 (10~15분)

사용자 직접 작업 (3분):
1. <https://app.heygen.com/signup> 가입
2. Settings → API → Create New Token
3. 토큰 복사 (`sk_V2_...`)
4. Claude 에 붙여넣기

Claude 자동 처리:
- `marketing-os/.env` 에 `HEYGEN_API_KEY=...` 추가
- `marketing-os/.mcp.json` 에 `heygen` 서버 항목 추가
- `uv tool install heygen-mcp` (사전 캐싱)
- API 통신 검증 (`/v2/voices` 응답 확인)

세션 재시작 (사용자 1회):
- 새 conversation 시작
- `mcp__heygen__*` 도구 활성화 확인

### Step 2 결과물: 60초 한국어 아바타 영상

```
"SunHi 보이스로 60초 한국어 마케팅 영상 만들어줘"

→ mcp__heygen__generate_avatar_video 호출
→ 1~3분 폴링 후 mp4 링크 반환
→ 다운로드 후 재생
```

### Step 3. ElevenLabs MCP 설치 (5~10분)

사용자 직접 작업 (3분):
1. <https://elevenlabs.io> 가입
2. Settings → API Keys → Create API Key
3. 토큰 복사 (`sk_...`)
4. Claude 에 붙여넣기

Claude 자동 처리:
- `marketing-os/.env` 에 `ELEVENLABS_API_KEY=...` 추가
- `marketing-os/.mcp.json` 에 `elevenlabs` 서버 항목 추가
- `uv tool install elevenlabs-mcp`
- 오디오 출력 폴더 생성: `hyperframes/assets/audio/`

세션 재시작 (사용자 1회):
- `mcp__elevenlabs__*` 도구 활성화 확인

### Step 3 결과물: 30초 한국어 내레이션 mp3

```
"다음 텍스트로 30초 한국어 내레이션 만들어줘:
 '이번 주 광고 성과를 정리합니다. ROAS 3.2배 ...'"

→ mcp__elevenlabs__text_to_speech 호출
→ hyperframes/assets/audio/narration.mp3 저장
→ 재생 후 발음 품질 확인
```

### Step 4. 통합 결과물: 53초 데이터 영상

```
"주간 ROAS 3.2배 데이터로 53초 영상 만들어줘.
 한국어 내레이션은 본인 음성으로."

→ /영상제작 스킬 자동 호출 (8단계)
→ Hyperframes HTML 작성 (sub-comp 8개)
→ ElevenLabs 내레이션 합성 → audio 트랙 추가
→ npm run check (lint + validate + inspect)
→ npm run render → 1920×1080 mp4 출력
→ open 명령으로 자동 재생
```

## 마케터 5패턴: 정기 운영 결합

```
[역할]
퍼포먼스 마케터의 영상 자동화 어시스턴트

[입력]
Google Sheets 의 지난주 광고 데이터 (광고비, ROAS, CTR, CPA)

[산출물]
53초 영상 (1920×1080 mp4) + 본인 음성 한국어 내레이션
→ Discord #marketing 채널 자동 발송
→ Notion "주간 광고 보고" 페이지 임베드

[제약]
- 폰트 · 색상은 본 브랜드 가이드 (Pretendard, 다크 + 그린 액센트)
- 한국어 발음 자연스러움 최우선
- 재현 가능한 결정성 임베드 (Linux 렌더에서도 동일 결과)

[검증]
- npm run check 전 단계 통과
- 영상 길이 53초 ± 1초
- 한국어 자막 동기화 ± 100ms
- 렌더 시간 32초 이내
```

## 트러블슈팅

| 증상 | 원인 | 해결 |
|---|---|---|
| `npx hyperframes doctor` 가 ffmpeg 없다고 표시 | macOS 기본 ffmpeg 미설치 | 본 실습의 ffmpeg 자동 설치 명령 실행 |
| HeyGen 영상 생성이 5분 이상 안 나옴 | 큐 적체 또는 잘못된 voice_id | `get_voices` 로 voice_id 재확인. 새벽 시간대 재시도 |
| ElevenLabs 한국어 발음 어색 | model 이 v1 또는 multilingual_v1 | `model_id=eleven_v3` 명시 |
| 세션 재시작 후에도 `mcp__heygen__*` 안 보임 | `.env` 환경 변수 미로드 또는 `.mcp.json` 문법 오류 | `python3 -c "import json; json.load(open('.mcp.json'))"` JSON 검증. `cat .env` 변수 확인 |
| 통합 영상에서 음성 트랙 동기 안 맞음 | mp3 길이와 영상 `data-duration` 불일치 | mp3 ffprobe 로 정확한 길이 측정 후 영상 duration 맞춤 |
