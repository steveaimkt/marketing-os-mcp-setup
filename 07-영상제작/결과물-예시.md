## 🧠 마케터 활용법 6가지

### A. SNS 일주일치 릴스 일괄 생성
```
"다음 5개 스크립트로 인스타 릴스 5개 만들어줘"
→ 트리오 5회 자동 실행 → Buffer 5개 채널 예약
```

### B. 강의·웨비나 영상
```
"이 슬라이드 + 대본으로 15분 강의 영상"
→ Hyperframes 슬라이드 변환 + ElevenLabs 본인 음성
```

### C. 신상품 광고 다국어
```
"이 한국어 광고를 영어·일본어·중국어로 더빙"
→ ElevenLabs 다국어 + HeyGen 립싱크
```

### D. 고객 후기 영상화
```
"고객 후기 텍스트 5개를 각각 15초 영상으로"
→ HeyGen 다양한 아바타 + 후기 음성
```

### E. 정기 KPI 비디오 리포트
```
"매주 광고 KPI 표를 영상 리포트로 (1분)"
→ Sheets 데이터 → Hyperframes 차트 + 음성 해설
```

### F. 본인 얼굴 출연 영상 (Personal Avatar)
```
"본인 얼굴 학습 → 본인이 직접 출연하는 영상"
→ HeyGen Personal Avatar + ElevenLabs 본인 클론
```

1. **Hyperframes**: `marketing-os/hyperframes/` 폴더에서 `npm install` (한 번만)
2. **HeyGen API key**: app.heygen.com → Settings → API → 발급 → `.env`에 `HEYGEN_API_KEY=` 등록
3. **ElevenLabs API key**: elevenlabs.io → Profile → API → 발급 → `.env`에 `ELEVENLABS_API_KEY=` 등록

## 💡 자주 묻는 질문

**Q. 본인 음성 클론 시간?**  
A. ElevenLabs 본인 녹음 5~10분 업로드 → 학습 약 1시간. 이후 영구 사용.

**Q. 한국어 자연스러움?**  
A. ElevenLabs Multilingual v2 모델 한국어 발음 매우 자연스러움 (TTS 평가 91~95점).

**Q. HeyGen 한국 아바타 종류?**  
A. KR1~KR15 (남녀·연령대 다양). 본인 얼굴 학습 (Personal Avatar) 도 가능.

**Q. 무료 한도?**  
A. HeyGen: 무료 0분/월 (유료부터 시작). ElevenLabs: 무료 10K 크레딧/월 (약 10분 음성). 본격 사용은 유료 권장.

**Q. 저작권 안전?**  
A. ElevenLabs 본인 클론은 본인 권리. HeyGen 기본 아바타는 라이선스 포함. 단 본인 클론으로 타인 사칭 금지.
