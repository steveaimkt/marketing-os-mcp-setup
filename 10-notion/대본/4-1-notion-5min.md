## 슬라이드 6장 + 5분 대본

### [0:00~0:25] Cover

> "Part 2 열 번째 도구는 Notion(노션) MCP입니다. **본인 노션 워크스페이스의 페이지·데이터베이스를 Claude(클로드)가 직접 만들고 검색하고 수정**해주는 도구예요.
>
> 본 클립이 끝나면 매주 콘텐츠 캘린더 30~50개 행을 손으로 입력하던 1시간이 5분에 끝납니다."

### [0:25~1:30] Before/After · 1시간 → 5분

> "1인 마케터의 노션 작업이 매우 많아요. 마케팅 데이터의 메타데이터가 다 노션에 모입니다.
>
> 첫째 **콘텐츠 캘린더 입력** · 매주 30~50개 카드를 채널·날짜·상태 포함해서 손으로 입력. 1~2시간.
>
> 둘째 **광고 리포트 정리** · 매주 ROAS 분석 결과를 노션 페이지로 복붙. 30분.
>
> 셋째 **VoC(브이오씨·고객의소리) 분석 결과 등록** · 시간당 약 20행 손 작업.
>
> 넷째 **회사 위키·매뉴얼 검색** · 흩어진 페이지에서 필요한 정보 찾기. 매번 5~10분.
>
> **Before(전)** · 매주 약 3시간 노션 손 작업.
>
> **After(후)** · Claude에 한 줄. '5월 콘텐츠 캘린더 30개 카드를 채널·날짜·상태 포함해서 등록.' 5분. 광고 리포트도 '이 분석 결과를 주간 리포트 페이지로 만들어줘.' 30초.
>
> 약 36배 단축. **월 12시간 절감.**"

### [1:30~3:00] 5 시나리오

> "Notion MCP로 자동화되는 5가지.
>
> **A. 워크스페이스 검색** · '봄 캠페인 관련 페이지' 5초.
> **B. DB 카드 일괄 등록 ★** · '5월 콘텐츠 캘린더 30개 카드' 5분. **가장 강력.**
> **C. 페이지 자동 생성 ★** · '이 광고 리포트를 주간 리포트 페이지로' 30초.
> **D. 페이지 수정** · '상태를 발행됨으로 변경' 5초.
> **E. 새 DB 생성** · '광고 리포트 아카이브 DB (날짜·매체·ROAS·인사이트)' 1분.
>
> 시작 순서 · 처음이면 A·D (검색·수정). 익숙해지면 C 페이지 생성. 매주 반복은 B 일괄 등록."

### 🎬 [3:00~4:00] 작동 원리 · 한 줄 명령이 콘텐츠 캘린더 30개 등록이 되기까지

〔슬라이드: 04 작동 흐름 · "notion 시작하자" → 라우터 → 메뉴 → 자동 실행 → 결과물〕

**🎙️ 강사 발화:**

> "실제로 어떻게 작동하는지 원리를 짚어드릴게요. 설치가 끝난 상태라면, Claude한테 단어 하나만 입력하면 됩니다.
>
> **`notion 시작하자`** ← 슬래시 명령 아니에요. 그냥 평문 한 문장. 본 강의가 미리 만들어둔 라우터가 자동으로 켜집니다.
>
> 그러면 Notion(노션)이 설치되어 있는 걸 자동으로 감지해서, **작업 가능 업무 5개 메뉴**를 띄워줍니다 · 워크스페이스 검색·페이지 자동 생성·페이지 일괄 수정·DB 카드 일괄 등록·새 DB 생성. 본인은 '**4**' 한 번만 누르면 돼요.
>
> 내부적으로는 이런 순서로 진행됩니다 · 노션 연결 확인 → 대상 DB 감지 → 30개 카드를 채널·날짜·상태에 맞춰 일괄 생성. 약 5분이면 노션 DB에 30개 행이 자동 등록됩니다.
>
> 결과물은 '5월 콘텐츠 캘린더' DB에 채널·날짜·상태가 채워진 30개 카드입니다. (실제 실행 화면은 별도 실습 영상에서 보여드립니다.)
>
> 다른 MCP와의 조합이 강력. 예 · Firecrawl로 경쟁사 데이터 수집 → Notion 광고 리포트 페이지 자동 생성. Sheets로 매출 분석 → Notion 주간 리포트 페이지 자동 발행."

### [4:00~4:35] 마케터 가치 3가지

> "Notion MCP의 가치 3가지.
>
> **첫째, 시간 절감** · 캘린더 입력 1시간 → 5분. 월 12시간 절감.
>
> **둘째, 메타데이터 허브** · 마케팅 모든 데이터가 노션 한 곳에 모임. 광고·콘텐츠·VoC·위키 통합 관리.
>
> **셋째, MCP 조합 시너지** · Firecrawl·Sheets·Ads와 연결 시 분석 → 노션 자동 발행 파이프라인.
>
> 한 줄 정리 · **노션이 마케팅의 자동 발행 허브가 된다.**"

### 🎬 [4:35~5:00] Closing · 설치는 Part 2 마지막 영상에서

〔슬라이드: 06 Sunset + Notion 실제 시연 영상 안내〕

**🎙️ 강사 발화:**

> "Notion의 개념과 작동 원리는 여기까지입니다.
>
> 설치는 따로 진행해요. 본 강의 Part 2 **마지막 영상**이 '설치 마라톤' 영상입니다. 11개 MCP 이론을 다 보신 후, 마지막 영상에서 Claude Code에 '**MCP 전체 설치하자**' 한 줄 입력하면 11개 MCP가 한 번에 자동 설치됩니다. 사전 준비물 같은 자세한 안내도 그 영상에서 함께 다룹니다.
>
> 자, 그럼 지금부터 **Notion 실제 시연 영상**을 보겠습니다. 방금 설명한 작동 원리가 실제 화면에서 어떻게 돌아가는지 직접 확인하실 수 있어요."
