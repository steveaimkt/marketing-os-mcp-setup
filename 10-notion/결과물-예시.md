## 🧠 마케터 활용법 6가지

### A. 주간/월간 리포트 자동 게시
```
"이번 주 리포트 Notion 'Weekly Reports' DB에 게시"
→ notion-create-pages + DB 속성 자동
```

### B. 콘텐츠 캘린더 관리
```
"다음 달 콘텐츠 계획 30개 Notion DB로 만들어줘"
→ notion-create-database + 30 rows
```

### C. 미팅 노트 자동 정리
```
"오늘 미팅 녹음 트랜스크립트를 Notion 미팅 노트로"
→ ElevenLabs 트랜스크립션 + notion-create-pages
```

### D. 경쟁사 모니터링 페이지
```
"Firecrawl 경쟁사 분석 결과를 Notion 'Competitor Watch' 페이지 추가"
→ Firecrawl + notion-update-page
```

### E. 캠페인 회고 아카이브
```
"종료된 캠페인 회고 페이지 자동 생성 (KPI + 인사이트 + 학습)"
→ notion-create-pages (캠페인 회고 템플릿)
```

### F. 팀 검색 가능 지식 베이스
```
"이번 분기 모든 분석 결과를 Notion 'Insights' DB에 자동 적재"
→ 매 분석 직후 notion-create-pages 자동 호출
```

## 🔁 marketing-os 전체 자동화의 보관소

Notion은 marketing-os 모든 결과물의 **장기 보관·검색 가능 지식 베이스**:

```
GA4 MCP → Notion 'Traffic' DB 적재
Sheets MCP → Notion 'Revenue' DB 적재
Ads MCP → Notion 'Ads' DB 적재
Firecrawl → Notion 'Competitor' DB 적재
Discord 알림 → Notion 'Activities' DB 적재
```

3개월 후 "Q1 인사이트 검색"으로 모든 데이터 즉시 찾기.

## 💡 자주 묻는 질문

**Q. Notion 무료 플랜?**  
A. 개인 무료 OK. 팀 협업은 Plus ($8/월) 권장.

**Q. 한국어 페이지 정상?**  
A. UTF-8 OK. 한글 제목·본문·DB 속성 정상.

**Q. 이미지·파일 첨부?**  
A. 가능. URL 또는 로컬 파일 전달.

**Q. 기존 페이지 업데이트만 (덮어쓰기 X)?**  
A. notion-update-page 도구로 가능. "X 페이지 마지막에 추가" 자연어 OK.

**Q. 데이터베이스 뷰 자동 생성?**  
A. notion-create-view 도구. 필터·정렬·갤러리 뷰 자동.
